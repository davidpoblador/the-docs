expiry (1)

Check and enforce password expiration policy

  1. Carta.tech
  2. Man Pages
  3. Executable programs or shell commands
  4. expiry

SYNOPSIS

expiry option

DESCRIPTION

The expiry command checks (-c) the current password expiration and forces (-f) changes when required. It is callable as a normal user command.

OPTIONS

The options which apply to the expiry command are:

-c, --check Check the password expiration of the current user.

-f, --force Force a password change if the current user has an expired password.

-h, --help Display help message and exit.

FILES

/etc/passwd User account information.

/etc/shadow Secure user account information.
