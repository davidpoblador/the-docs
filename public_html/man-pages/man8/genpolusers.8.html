genpolusers (8)

Generate new binary policy with updated user configuration

  1. Carta.tech
  2. Man Pages
  3. System administration commands
  4. genpolusers

SYNOPSIS

genpolusers in-policy usersdir out-policy

DESCRIPTION

Given an existing binary policy file in-policy, generate a new binary policy out-policy with an updated user configuration based on any system.users and local.users files in the specified usersdir.
