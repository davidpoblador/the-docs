selinux-polgengui (8)

Selinux policy generation tool

  1. Carta.tech
  2. Man Pages
  3. System administration commands
  4. selinux-polgengui

SYNOPSIS

selinux-polgengui

DESCRIPTION

selinux-polgengui is a graphical tool, which can be used to create a framework for building SELinux Policy.

OPTIONS

None

FILES

\fi/usr/bin/selinux-polgengui

Examples

To run the program type:

selinux-polgengui

selinux(1), sepolicy(8), sepolicy-generate(8)
