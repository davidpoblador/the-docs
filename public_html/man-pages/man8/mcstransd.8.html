mcstransd (8)

Mcs (multiple category system) daemon. translates selinux mcs/mls labels to human readable form.

  1. Linux Man Pages
  2. System administration commands
  3. mcstransd

SYNOPSIS

mcstransd [-f] [-h]

DESCRIPTION

This manual page describes the mcstransd program.

This daemon reads /etc/selinux/{SELINUXTYPE}/setrans.conf configuration file, and communicates with libselinux via a socket in /var/run/setrans.

OPTIONS

-f

Run mcstransd in the foreground. Do not run as a daemon.

-h

Output a short summary of available command line options.
