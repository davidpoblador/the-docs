C SYNOPSIS

#include <pcp/pmapi.h>

int pmWhichContext(void);

cc ... -lpcp

DESCRIPTION

An application using the Performance Metrics Application Programming Interface (PMAPI) may manipulate several concurrent contexts, each associated with a source of performance metrics, e.g. .BR pmcd (1) on some host, or an archive log of performance metrics as created by pmlogger(1).

pmWhichContext returns a handle for the current PMAPI context, that may be used in the associated PMAPI routines that require a handle to identify a PMAPI context.

RELATED TO pmwhichcontext…

PMAPI(3), pmDestroyContext(3), pmDupContext(3), pmNewContext(3) and pmUseContext(3).

DIAGNOSTICS
