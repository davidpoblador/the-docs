koi8-r (7)

Russian character set encoded in octal, decimal, and hexadecimal

  1. Linux Man Pages
  2. Miscellaneous
  3. koi8-r

DESCRIPTION

RFC 1489 defines an 8-bit character set, KOI8-R. KOI8-R encodes the characters used in Russian.
