mq_getsetattr (2)

Get/set message queue attributes

  1. Carta.tech
  2. man-pages
  3. System calls
  4. mq_getsetattr

SYNOPSIS

#include <sys/types.h>
#include <mqueue.h>

int mq_getsetattr(mqd_t mqdes, struct mq_attr *newattr,
                 struct mq_attr *oldattr);

Note: There is no glibc wrapper for this system call; see NOTES.

DESCRIPTION

Do not use this system call.

This is the low-level system call used to implement mq_getattr(3) and mq_setattr(3). For an explanation of how this system call operates, see the description of mq_setattr(3).

CONFORMING TO

This interface is nonstandard; avoid its use.

NOTES

Glibc does not provide a wrapper for this system call; call it using syscall(2). (Actually, never call it unless you are writing a C library!)
