SYNOPSIS

  my $perlanet = Perlanet->new_with_traits(
    traits => [ 'Perlanet::Trait::OPML' ]
  );

  $perlanet->run;

DESCRIPTION

Generates an \s-1OPML\s0 file of all feeds that are being aggregated by the planet.

ATTRIBUTES

opml_generator

An XML::OPML::SimpleGen object to generate the \s-1XML\s0 for the \s-1OPML\s0 file

opml_file

Where to save the \s-1OPML\s0 feed when it has been created

METHODS

update_opml

Updates the \s-1OPML\s0 file of all contributers to this planet. If the opml attribute does not have a value, this method does nothing, otherwise it inserts each author into the \s-1OPML\s0 file and saves it to disk.

save_opml

Save the \s-1OPML\s0 file, by default to disk.
