SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'da_DK' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Danish Denmark.

DATA

This locale inherits from the DateTime::Locale::da locale.

It contains the following data.
