DESCRIPTION

Net::NBName::NameQuery::RR represents an ip address entry returned as part of a NetBIOS name query response.

METHODS

$rr->address

Returns the ip address as a dotted quad.

$rr->G

Group flag. Indicates whether the name is a unique or a group name. It is returned as a string: either \*(L"\s-1UNIQUE\s0\*(R" or \*(L"\s-1GROUP\s0\*(R" will be returned.

$rr->\s-1ONT\s0

Owner Node Type flag. Indicates if the systems are B, P, H, or M-node. It will be returned as a string.

$rr->as_string

Returns the object's string representation.
