DESCRIPTION

The Algorithm::Dependency::Item class implements a single item within the dependency heirachy. It's quite simple, usually created from within a source, and not typically created directly. This is provided for those implementing their own source. ( See Algorithm::Dependency::Source for details ).

METHODS

The \*(C`new\*(C' constructor takes as its first argument the id ( name ) of the item, and any further arguments are assumed to be the ids of other items that this one depends on.

Returns a new \*(C`Algorithm::Dependency::Item\*(C' on success, or \*(C`undef\*(C' on error.

id

The \*(C`id\*(C' method returns the id of the item.

depends

The \*(C`depends\*(C' method returns, as a list, the names of the other items that this item depends on.
