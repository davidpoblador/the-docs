DESCRIPTION

Non-empty inline elements that can be included within a HTML document. Example open tags of the HTML elements produced are below - see the

HTML specifications

for their semantics and uses.

  • - Emphasis - <em>

  • - StrongEmphasis - <strong>

  • - Abbreviation("Algebraic Data Type") - <abbr title="Algebraic Data Type">

  • - Citation - <cite>

  • - Definition - <dfn>

  • - ComputerCode - <code>

  • - SampleInput - <kbd>

  • - SampleOutput - <samp>

  • - Variable - <var>

  • - BiggerText - <big>

  • - SmallerText - <small>

  • - Bold - <b>

  • - Italic - <i>

  • - Subscript - <sub>

  • - Superscript - <sup>

  • - Hyperlink("http://www.example.com/") - <a href="http://www.example.com/">

  • - InlineQuote("http://www.example.com/") - <q cite="http://www.example.com/">

  • - Span("note") - <span class="note">

  • - FormLabel - <label>
