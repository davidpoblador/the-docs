dssetc (3NCARG)

Sets the value of an internal parameter of type character.

SYNOPSIS

CALL DSSETC (PNAM, CVAL);

DESCRIPTION

PNAM

A character string that specifies the name of the parameter to be set. The name must appear as the first three characters of the string.

CVAL

A character string that is the value to be assigned to the internal parameter specified by pnam.

USAGE

This function allows you to set the current value of Dsgrid parameters. For a complete list of parameters available in this utility, see the dsgrid_params man page.

ACCESS

To use DSSETC, load the NCAR Graphics library ngmath.
