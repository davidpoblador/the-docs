Constructor & Destructor Documentation

ost::IPV4Host::IPV4Host (const char *host = \fCNULL\fP)

Create a new host address for a specific internet host. The internet host can be specified in a null terminated ASCII string and include either the physical host address or the DNS name of a host machine. Hence, an IPV4Host ('www.voxilla.org') can be directly declaired in this manner.

Defaults to the IP address that represents the interface matching 'gethostname()'.

Parameters:

host dns or physical address of an Internet host.

ost::IPV4Host::IPV4Host (struct in_addraddr)

Convert a system socket binary address such as may be returned through the accept() call or getsockpeer() into an internet host address object.

Parameters:

addr binary address of internet host.
