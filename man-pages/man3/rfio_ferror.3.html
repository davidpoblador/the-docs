SYNOPSIS

#include <sys/types.h>

#include "rfio_api.h"

int rfio_ferror (FILE *fp);

int rfio_feof (FILE *fp);

DESCRIPTION

rfio_ferror tests the error indicator for the file whose descriptor fp is the one returned by rfio_fopen.

rfio_feof tests the end-of-file indicator for the file whose descriptor fp is the one returned by rfio_fopen.

RETURN VALUE

This routine returns >0 if set, 0 if not set, -1 if the operation failed and serrno is set appropriately.

ERRORS

EBADF

fp is not a valid file descriptor.

SEBADVERSION

Version ID mismatch.
