SYNOPSIS

  use Business::EDI::Spec;

  my $spec = Business::EDI::Spec->new('segment');

DESCRIPTION

\s-1CSV\s0 files originally from edi4r are included as part of Business::EDI. They are used to define the many different messages, segements, data elements, composite data elements, and codelists that are part of a given version of the U.N. specification.

The \s-1CSV\s0 spec files are composed differently for the different structures defined. So we have to parse them differently.
