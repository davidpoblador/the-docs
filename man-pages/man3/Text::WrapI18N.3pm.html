Text::WrapI18N (3pm)

Line wrapping module with support for multibyte, fullwidth, and combining characters and languages without whitespaces between words

SYNOPSIS

  use Text::WrapI18N qw(wrap $columns);
  wrap(firstheader, nextheader, texts);

DESCRIPTION

This module intends to be a better Text::Wrap module. This module is needed to support multibyte character encodings such as \s-1UTF-8\s0, EUC-JP, EUC-KR, \s-1GB2312\s0, and Big5. This module also supports characters with irregular widths, such as combining characters (which occupy zero columns on terminal, like diacritical marks in \s-1UTF-8\s0) and fullwidth characters (which occupy two columns on terminal, like most of east Asian characters). Also, minimal handling of languages which doesn't use whitespaces between words (like Chinese and Japanese) is supported.

Like Text::Wrap, hyphenation and \*(L"kinsoku\*(R" processing are not supported, to keep simplicity.

wrap(firstheader, nextheader, texts) is the main subroutine of Text::WrapI18N module to execute the line wrapping. Input parameters and output data emulate Text::Wrap. The texts have to be written in locale encoding.
