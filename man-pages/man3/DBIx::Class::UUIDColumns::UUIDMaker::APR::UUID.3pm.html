SYNOPSIS

  package Artist;
  _\|_PACKAGE_\|_->load_components(qw/UUIDColumns Core DB/);
  _\|_PACKAGE_\|_->uuid_columns( 'artist_id' );
  _\|_PACKAGE_\|_->uuid_class('::APR::UUID');

DESCRIPTION

This DBIx::Class::UUIDColumns::UUIDMaker subclass uses \s-1APR::UUID\s0 to generate uuid strings in the following format:

098f2470-bae0-11cd-b579-08002b30bfeb

METHODS

as_string

Returns the new uuid as a string.
