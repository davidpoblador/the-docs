SYNOPSIS

  use Pod::PseudoPod::Checker;

  my $checker = Pod::PseudoPod::Checker->new();

  ...

  $checker->parse_file('path/to/file.pod');

DESCRIPTION

This class is for checking the syntactic validity of Pod. It works by basically acting like a simple-minded version of Pod::PseudoPod::Text that formats only the \*(L"Pod Errors\*(R" section (if Pod::PseudoPod even generates one for the given document). It's largely unchanged from Pod::Simple::Checker.
