SYNOPSIS

   my $perlanet = Perlanet->new_with_traits(
     traits => [ 'Perlanet::Trait::FeedFile' ]
   );

   $perlanet->run;

DESCRIPTION

When the aggregation is complete and the feed is being rendered, it will be saved to disk in \s-1XML\s0 format.

ATTRIBUTES

feed_file

The path to the file to save the feed to.

feed_format

The format of the \s-1XML\s0 to use - may be \s-1RSS\s0 or Atom
