SYNOPSIS

 use Icon::FamFamFam::Silk;
 my $wrench = Icon::FamFamFam::Silk->new('wrench_orange');
 print $wrench->base64;

or

use Icon::FamFamFam::Silk; print Icon::FamFamFam::Silk::base64('wrench_orange');

DESCRIPTION

These methods can be called as functions taking the name of the icon (a string) as their argument, or can be called as object methods on an instantiated icon object. (See \*(L"\s-1SYNOPSIS\s0\*(R".) Returns a blessed icon object. Returns a list of all the icons. Returns the name of the icon. Returns base64-encoded \s-1PNG\s0 data. Returns raw \s-1PNG\s0 data. Returns a \*(L"data:\*(R" \s-1URI\s0 for the icon. Returns a hash of data useful for giving credit to Mark James. This is in a format suitable for passing to HTML::HTML5::Builder's \*(C`CREATIVE_COMMONS\*(C' function, should you want to do something like that.
