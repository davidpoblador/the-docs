SYNOPSIS

        use NetSDS::Const;

        print "XML encoding is " . XML_ENCODING;

        print "Week contains " . INTERVAL_WEEK . " seconds";

DESCRIPTION

This module provides most common constants like default encoding and language, time intervals, etc.

LANGUAGE AND ENCODINGS

XML CONSTANTS

TIME INTERVALS

\s-1INTERVAL_MINUTE\s0 - 60 seconds
\s-1INTERVAL_HOUR\s0 - 3600 seconds
\s-1INTERVAL_DAY\s0 - 86400 seconds
\s-1INTERVAL_WEEK\s0 - 604800 seconds
