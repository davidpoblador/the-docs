VERSION

This documentation refers to CGI::Application::Plugin::DebugScreen version 1.00

SYNOPSIS

  use CGI::Application::Plugin::DebugScreen;

That is it! The fun starts when a run mode calls \*(C`die\*(C'.

DESCRIPTION

This plug-in adds stack tracing support to CGI::Application similar in style to the Catalyst debug mode. In the event of the module calling \*(C`die\*(C', a stack trace of the error is displayed. At each level of the stack links to the code and documentation are provided.

The stack trace functionality is deactivated unless the $ENV{\s-1CGI_APP_DEBUG\s0} environment variable is set. Therefore when your code is released, this plug-in does not need to be removed.

The links links to code and documentation are implemented using CGI::Application::Plugin::ViewCode, which will be loaded automatically if available. However these links are not supported in conjunction with CGI::Application::Dispatch.
