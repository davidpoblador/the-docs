DESCRIPTION

The \s-1HTML-4\s0.0 specification defines 16 color names assigned to the full and half coordinate \s-1RGB\s0 primaries.

http://www.w3.org/TR/REC-html40/sgml/loosedtd.html#Color <http://www.w3.org/TR/REC-html40/sgml/loosedtd.html#Color>
