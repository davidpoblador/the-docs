SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'it_CH' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Italian Switzerland.

This locale inherits from the DateTime::Locale::it locale.
