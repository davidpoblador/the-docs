SYNOPSIS

#include <sys/types.h>

#include "lfc_api.h"

int lfc_startsess (char *server, char *comment);

DESCRIPTION

lfc_startsess starts session. The connection to the Name Server is kept open until lfc_endsess is explicitly called. Requests issued between lfc_startsess and lfc_endsess are independent from each other and are individually committed or rolled back. The comment is logged in the server log.

RETURN VALUE

This routine returns 0 if the operation was successful or -1 if the operation failed. In the latter case, serrno is set appropriately.

ERRORS

EINVAL

The length of comment exceeds CA_MAXCOMMENTLEN.

SENOSHOST

Host unknown.

SENOSSERV

Service unknown.

SECOMERR

Communication error.

ENSNACT

Name server is not running or is being shutdown.
