DESCRIPTION

This module represents a remote iPhoto shared album.

METHODS

count

The returns the number of images in the album.

id

This returns the internal iPhoto \s-1ID\s0 for the album. You probably don't need to worry about this.

images

This returns an arrayref of Net::DPAP::Client::Image objects, representing the images in the album.

name

This returns the name of the album. Note that if you are sharing individual albums, iPhoto tends to share all the images in the collection in an album named \*(L"Photo album\*(R", as well as in the individual albums. So you may see photos twice in that case.
