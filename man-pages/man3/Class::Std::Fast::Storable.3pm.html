VERSION

This document describes Class::Std::Fast::Storable 0.0.8

SYNOPSIS

    package MyClass;

    use Class::Std::Fast::Storable;

    1;

    package main;

    use Storable qw(freeze thaw);

    my $thawn = freeze(thaw(MyClass->new()));

DESCRIPTION

Class::Std::Fast::Storable does the same as Class::Std::Storable does for Class::Std. The \s-1API\s0 is the same as Class::Std::Storable's, with few exceptions.
