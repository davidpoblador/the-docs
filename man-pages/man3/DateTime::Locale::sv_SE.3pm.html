SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'sv_SE' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Swedish Sweden.

This locale inherits from the DateTime::Locale::sv locale.
