Module Id : sig end

Node identifiers

type +'a id

The type of global SVG element identifier.

val new_elt_id : ?global:bool -> unit -> 'a id

The function new_elt_id () creates a new HTML5 element identifier. (see the Eliom manual for more information on ).

val create_named_elt : id:'a id -> 'a Eliom_content.Svg.elt -> 'a Eliom_content.Svg.elt

The function create_named_elt ~id elt create a copy of the element elt that will be accessible through the name id .

val create_global_elt : 'a Eliom_content.Svg.elt -> 'a Eliom_content.Svg.elt

The function create_named_elt elt is equivalent to create_named_elt ~id:(new_elt_id ()) elt .

val get_element : 'a id -> 'a Eliom_content.Svg.elt

get_element id returns the HTML element in the DOM with the given id .
