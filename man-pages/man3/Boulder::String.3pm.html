SYNOPSIS

   #!/bin/perl
   # Read a B<Stone> from stdin and create a string that can be
   # passed to a dumb sub, which doesn't know about Stones.
   use Boulder::Stream;
   use Boulder::String;

   my $stream = Boulder::Stream->newFh;

   # read a stone from stdin
   my $record = <$stream> );

      print $stream $record;
    }

DESCRIPTION

Boulder::String provides access to Boulder \s-1IO\s0 hierarchical tag/value data. Stone objects printed to the tied string are appended to the string in Boulder format.

Boulder::Stream METHODS

The new() method creates a new Boulder::String object. You must provide an input string and a reference to an output string. The input string may be empty. Write the passed stone in Boulder \s-1IO\s0 format into $out_string.
