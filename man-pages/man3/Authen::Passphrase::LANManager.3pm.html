DESCRIPTION

An object of this class encapsulates a passphrase hashed using the Microsoft \s-1LAN\s0 Manager hash function. This is a subclass of Authen::Passphrase, and this document assumes that the reader is familiar with the documentation for that class.

The hash algorithm can be used on up to fourteen Latin-1 characters of passphrase. First the passphrase is folded to uppercase, and zero-padded to fourteen bytes. Then it is split into two halves. Each seven-byte half is used as a 56-bit \s-1DES\s0 key, to encrypt the fixed plaintext block \*(L"\s-1KGS\s0!@#$%\*(R". The eight-byte ciphertexts are concatenated to form the sixteen-byte hash. There is no salt.

Because the two halves of the passphrase are hashed separately, it is possible to manipulate (e.g., crack) a half hash in isolation. See Authen::Passphrase::LANManagerHalf.

Warning: Don't even think about using this seriously. It's an exceptionally weak design, flawed in pretty much every respect.

CONSTRUCTORS

Authen::Passphrase::LANManager->new(\s-1ATTR\s0 => \s-1VALUE\s0, ...)

Generates a new passphrase recogniser object using the \s-1LAN\s0 Manager hash algorithm. The following attributes may be given:

hash

The hash, as a string of 16 bytes.

hash_hex

The hash, as a string of 32 hexadecimal digits.

passphrase

A passphrase that will be accepted.

Either the hash or the passphrase must be given.

Authen::Passphrase::LANManager->from_rfc2307(\s-1USERPASSWORD\s0)

Generates a \s-1LAN\s0 Manager passphrase recogniser from the supplied \s-1RFC2307\s0 encoding. The string must consist of "{\s-1LANMAN\s0}\*(L" (or its synonym \*(R"{\s-1LANM\s0}") followed by the hash in hexadecimal; case is ignored.

METHODS

Returns the hash value, as a string of 16 bytes. Returns the hash value, as a string of 32 hexadecimal digits. Returns the hash of the first half of the passphrase, as an Authen::Passphrase::LANManagerHalf passphrase recogniser. Returns the hash of the second half of the passphrase, as an Authen::Passphrase::LANManagerHalf passphrase recogniser. These methods are part of the standard Authen::Passphrase interface.
