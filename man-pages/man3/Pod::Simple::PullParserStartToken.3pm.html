DESCRIPTION

When you do $parser->get_token on a Pod::Simple::PullParser object, you might get an object of this class.

This is a subclass of Pod::Simple::PullParserToken and inherits all its methods, and adds these methods: This returns the tagname for this start-token object. For example, parsing a \*(L"=head1 ...\*(R" line will give you a start-token with the tagname of \*(L"head1\*(R", token(s) for its content, and then an end-token with the tagname of \*(L"head1\*(R". This changes the tagname for this start-token object. You probably won't need to do this. A shortcut for $token->tagname(...) These are shortcuts for \*(C`$token->tag() eq somestring\*(C' This returns the value of the attrname attribute for this start-token object, or undef. For example, parsing a L<Foo/\*(L"Bar\*(R"> link will produce a start-token with a \*(L"to\*(R" attribute with the value \*(L"Foo\*(R", a \*(L"type\*(R" attribute with the value \*(L"pod\*(R", and a \*(L"section\*(R" attribute with the value \*(L"Bar\*(R". This sets the attrname attribute for this start-token object to newvalue. You probably won't need to do this. This returns the hashref that is the attribute set for this start-token. This is useful if (for example) you want to ask what all the attributes are \*(-- you can just do \*(C`keys %{$token->attr_hash}\*(C'

You're unlikely to ever need to construct an object of this class for yourself, but if you want to, call \*(C`Pod::Simple::PullParserStartToken->new( tagname, attrhash )\*(C'
