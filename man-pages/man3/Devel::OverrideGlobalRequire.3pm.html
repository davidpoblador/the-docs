VERSION

version 0.001

SYNOPSIS

  use Devel::OverrideGlobalRequire;

  override_global_require( sub { ... } );

DESCRIPTION

This module overrides \*(C`CORE::GLOBAL::require\*(C' with a code reference in a way that plays nice with any existing overloading and ensures the right calling package is in scope.
