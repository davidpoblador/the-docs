DESCRIPTION

This module contains methods for the parsing of reference metadata into OpenURLs. Although we have aimed to make it 1.0 compliant, there may well be errors (please let us know if there are!).

METHODS

This method takes a hash of metadata, and returns a hash that contains only valid OpenURL fields. This method aims to enrich an OpenURL metadata hash by applying various parsing techniques to the fields. It decomposes dates into years, months, and days if possible, fills in the appropriate fields if SICIs are present, and ensures URLs, ISBNs, etc, are valid. It returns a pointer to a hash containing the modified metadata, and an array of errors (if any). This method creates and returns an OpenURL from a metadata hash. No base URLs are prepended to this, so this should be done before using it as a link. URI::OpenURL should be used to generate OpenURLs in place of this function. This is a simple function that takes an array of \s-1ISBN\s0 characters, and returns true if it is a valid \s-1ISBN\s0. This function takes a \s-1SICI\s0 string, and returns a hash of information parsed from it, including date information, issn numbers, etc. This is not yet implemented, but will eventually be the \s-1BICI\s0 alternative for parse_sici. Simple function to convert a string into an encoded \s-1URL\s0 (i.e. spaces to %20, etc). Takes the unencoded \s-1URL\s0 as a parameter, and returns the encoded version.
