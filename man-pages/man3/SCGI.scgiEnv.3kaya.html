SYNOPSIS

String scgiEnv( String key )

ARGUMENTS

key Name of the environment variable

DESCRIPTION

Since environment variables are handled differently under the SCGI protocol to the conventional manner, this function must be used instead of System.getEnv (3kaya) to read environment variables within an SCGI application.
