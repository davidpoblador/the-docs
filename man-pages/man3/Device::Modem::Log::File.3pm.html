SYNOPSIS

  use Device::Modem;

my $box = Device::Modem->new( log => 'file', ... ); my $box = Device::Modem->new( log => 'file,name=/var/log/mymodem.log', ... ); ...

DESCRIPTION

This is meant for an example log class to be hooked to \*(C`Device::Modem\*(C' to provide one's favourite logging mechanism. You just have to implement your own \*(C`new()\*(C', \*(C`write()\*(C' and \*(C`close()\*(C' methods.

Default text file is \*(C`/var/log/modem.log\*(C'. On Windows platforms, this goes into \*(C`%WINBOOTDIR%/temp/modem.log\*(C'. By default, if the folder of the log file does not exist, it is created.

This class is loaded automatically by \*(C`Device::Modem\*(C' class when an object is instantiated, and it is the default logging mechanism for \*(C`Device::Modem\*(C' class.

Normally, you should not need to use this class directly, because there are many other zillions of modules that do logging better than this.

Also, it should be pondered whether to replace \*(C`Device::Modem::Log::File\*(C' and mates with those better classes in a somewhat distant future.
