SYNOPSIS

        use NetSDS::Template;

DESCRIPTION

\*(C`NetSDS::Template\*(C' class provides developers with ability to create template based web applications.

CLASS API

new([%params]) - class constructor

This concstructor loads template files and parse them. my $tpl = NetSDS::Template->new( dir => '/etc/NetSDS/templates', esc => 'URL', include_path => '/mnt/floppy/templates', ); This method prepares set of parameters and applies them to given template. Return is a ready for output document after processing. Example: # Simple template rendering with scalar parameters my $str = $tmp->render('main', title => 'Main Page');

# Rendering template with array parameters my $str2 = $tmp->render('list', title => 'Statistics', users => \@users_list);
