METHODS PROPERTIES

class

Returns the object class

string

Returns the \s-1JSON\s0 string

parent

Return the parent Net::GPSD object

device

tag

time

timestamp

datetime

reported

Count of satellites in view

used

Count of satellites used in calculation

satellites

Returns a list of satellite data structures.

my $satellites=$sky->satellites(); #[{},...] my @satellites=$sky->satellites(); #({},...)

Satellites

Returns a list of Net::GPSD3::Return::Satellite objects.

my @satellites=$sky->Satellites; #(bless{},...) my $satellites=$sky->Satellites; #[bless{},...]
