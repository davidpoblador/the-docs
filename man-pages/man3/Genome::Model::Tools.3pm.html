DESCRIPTION

Modules with names starting with Genome::Model::Tools are directly invokable by the \*(L"genome\*(R" command, installed with the core Genome package.

These modules all inherit from Command and use \s-1UR\s0 for their class definitions.

  • Copy the Example1.pm.template to Example1.pm and try \*(L"gmt\*(R" in the same directory for an example tool.

  • See Command for details on how to write commands.

  • See \s-1UR\s0 for details on writing \s-1UR\s0 classes in general.
