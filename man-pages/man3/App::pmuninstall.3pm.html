DESCRIPTION

App::pmuninstall is a fast module uninstaller. delete files from .packlist.

App::cpanminus and, App::cpanoutdated with a high affinity.

SYNOPSIS

uninstall \s-1MODULE\s0

  $ pm-uninstall App::pmuninstall

OPTIONS

-f, --force

Uninstalls without prompts $ pm-uninstall -f App::pmuninstall

-v, --verbose

Turns on chatty output $ pm-uninstall -v App::cpnaminus

-c, --checkdeps

Check dependencies ( default on ) $ pm-uninstall -c Plack

-n, --no-checkdeps

Don't check dependencies $ pm-uninstall -n LWP

-q, --quiet

Suppress some messages $ pm-uninstall -q Furl

-h, --help

Show help message $ pm-uninstall -h

-V, --version

Show version $ pm-uninstall -V

-l, --local-lib

Additional module path $ pm-uninstall -l extlib App::pmuninstall

-L, --local-lib-contained

Additional module path (don't include non-core modules) $ pm-uninstall -L extlib App::pmuninstall
