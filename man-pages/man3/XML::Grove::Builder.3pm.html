SYNOPSIS

 use PerlSAXParser;
 use XML::Grove::Builder;

 $builder = XML::Grove::Builder->new();
 $parser = PerlSAXParser->new( Handler => $builder );

 $grove = $parser->parse( Source => [SOURCE] );

DESCRIPTION

\*(C`XML::Grove::Builder\*(C' is a PerlSAX handler for building an XML::Grove.

\*(C`XML::Grove::Builder\*(C' is used by creating a new instance of \*(C`XML::Grove::Builder\*(C' and providing it as the Handler for a PerlSAX parser. Calling `\*(C`parse()\*(C'' on the PerlSAX parser will return the grove built from that parse.
