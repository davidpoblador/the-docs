SYNOPSIS

    use Lingua::StopWords qw( getStopWords );
    my $stopwords = getStopWords('en');

    my @words = qw( i am the walrus goo goo g'joob );

    # prints "walrus goo goo g'joob"
    print join ' ', grep { !$stopwords->{$_} } @words;

DESCRIPTION

In keyword search, it is common practice to suppress a collection of \*(L"stopwords\*(R": words such as \*(L"the\*(R", \*(L"and\*(R", \*(L"maybe\*(R", etc. which exist in in a large number of documents and do not tell you anything important about any document which contains them. This module provides such \*(L"stoplists\*(R" in several languages.

FUNCTIONS

getStopWords

my $stoplist = getStopWords('en'); my $utf8_stoplist = getStopWords('en', 'UTF-8');

Retrieve a stoplist in the form of a hashref where the keys are all stopwords and the values are all 1.

$stoplist = { and => 1, if => 1, # ... };

getStopWords() expects 1-2 arguments. The first, which is required, is an \s-1ISO\s0 code representing a supported language. If the \s-1ISO\s0 code cannot be found, getStopWords returns undef.

The second argument should be '\s-1UTF-8\s0' if you want the stopwords encoded in \s-1UTF-8\s0. The \s-1UTF-8\s0 flag will be turned on, so make sure you understand all the implications of that.
