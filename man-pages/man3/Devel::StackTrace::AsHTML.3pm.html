SYNOPSIS

  use Devel::StackTrace::AsHTML;

  my $trace = Devel::StackTrace->new;
  my $html  = $trace->as_html;

DESCRIPTION

Devel::StackTrace::AsHTML adds \*(C`as_html\*(C' method to Devel::StackTrace which displays the stack trace in beautiful \s-1HTML\s0, with code snippet context and function parameters. If you call it on an instance of Devel::StackTrace::WithLexicals, you even get to see the lexical variables of each stack frame.
