FUNCTION PROTOTYPE

void c_dsgeti(char *, int *);

SYNOPSIS

c_dsgeti (char *pnam, int *ival)

DESCRIPTION

pnam

A string that specifies the name of the parameter to get. The name must appear as the first three characters of the string.

ival

*ival will be the value currently assigned to the control parameter whose name is pointed to by pnam.

USAGE

This routine allows you to retrieve the current value of Dsgrid parameters. For a complete list of parameters available in this utility, see the dsgrid_params man page.

ACCESS

To use c_dsgeti, load the NCAR Graphics library ngmath.
