SYNOPSIS

    [% USE gd_text = GD.Text %]

EXAMPLES

[% USE gd_c = GD.Constants; USE t = GD.Text; x = t.set_text('Some text'); r = t.get('width', 'height', 'char_up', 'char_down'); r.join(":"); "\n"; # returns 54:13:13:0. -%]

[% USE gd_c = GD.Constants; USE t = GD.Text(text => 'FooBar Banana', font => gd_c.gdGiantFont); t.get('width'); "\n"; # returns 117. -%]

DESCRIPTION

The \s-1GD\s0.Text plugin provides an interface to the GD::Text module. It allows attributes of strings such as width and height in pixels to be computed.

See GD::Text for more details. See Template::Plugin::GD::Text::Align and Template::Plugin::GD::Text::Wrap for plugins that allow you to render aligned or wrapped text in \s-1GD\s0 images.
