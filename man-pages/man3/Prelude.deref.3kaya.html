SYNOPSIS

a deref( Maybe<a> v )

ARGUMENTS

v The value to dereference

DESCRIPTION

Extract the value from a Prelude.Maybe (3kaya) An Exception is thrown if v is nothing A case statemement is generally more appropriate unless it is known that the dereferencing should succeed (for example, using Dict.lookup (3kaya) on a key known to have been added). deref(x) is equivalent to x.val
