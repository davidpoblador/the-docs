SYNOPSIS

 use Net::Whois::RIPE;
 use Net::Whois::RIPE::Iterator;

 $whois = Net::Whois::RIPE->new($host);

 $iterator = Net::Whois::RIPE::Iterator->new($whois,$query);
 while ($obj = $iterator->next) {
     ...
 }

DESCRIPTION

Implements an iterator for Net::Whois::RIPE::Objects.

CONSTRUCTOR

new (\s-1WHOIS\s0, \s-1QUERY\s0)

The constructor returns undef on failure. If debug is on then a message is carped about the failure. \s-1WHOIS\s0 is an object of class Net::Whois::RIPE \s-1QUERY\s0 is a query string to be sent to the \s-1WHOIS\s0 object to retrieve the whois objects.

METHODS

next()

Returns the next Net::Whois::RIPE::Object from the query. Returns undef if no more objects.

debug(\s-1LEVEL\s0)

Sets/gets debugging level on the class or an object. 0 - no debugging 1 - debugging on
