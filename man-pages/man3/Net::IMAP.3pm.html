This is a container for \*(C`exists\*(C' responses. Returns the quantity of messages in the currently selected mailbox.

This is information is also available in the \*(C`qty_messages\*(C' method in the \*(C`Net::IMAP\*(C' class.

Recent

This is a container for \*(C`recent\*(C' responses. Returns the number of messages with the \*(C`\recent\*(C' flag set.

This information is also available in the \*(C`qty_recent\*(C' method in the \*(C`Net::IMAP\*(C' class.

Namespace

This is a container for \*(C`namespace\*(C' responses. With no argument specified, returns a list of personal namespaces. If $namespace is specified, returns the delimiter character for the specific personal namespace. With no argument specified, returns a list of other users' namespaces. If $namespace is specified, returns the delimiter character for the specific other users' namespace. With no argument specified, returns a list of shared namespaces. If $namespace is specified, returns the delimiter character for the specific shared namespace.

ACL

This is a container for \*(C`acl\*(C' responses> Returns the name of the mailbox associated with the given \s-1ACL\s0 data. Returns a list of identifiers contained in the \s-1ACL\s0 data.

Listrights

This is a container for \*(C`listrights\*(C' responses. Returns the name of the mailbox associated with the given rights. Returns a string containing the identifier associated with the rights. Returns a string containing the rights contained in the response.

Myrights

This is a container for \*(C`myrights\*(C' responses Returns the name of the mailbox associated with the given rights. Returns a string containing the rights contained in the response.

Quota

This is a container for \*(C`quota\*(C' responses. Returns a string containing the name of the quota root in the response. Returns a list of the quotas contained in the response. Returns the usage value associated with the given $quota. Returns \*(C`undef\*(C' is the given $quota is not present in the response. Returns the usage limit associated with the given $quota. Returns \*(C`undef\*(C' is the given $quota is not present in the response.

Quotaroot

This is a container for \*(C`quotaroot\*(C' responses. Returns the name of the mailbox associated with the quotaroot data. If called in an array context, returns the list of quotaroots associated with the mailbox. If called in a scalar context, returns a list reference.

MISC FETCH OBJECTS

A \*(C`fetch\*(C' response can be relatively complicated. This section documents various classes and methods associated with the various pieces of information available in \*(C`fetch\*(C' responses.

BodyStructure

This is a container for \*(C`bodystructure\*(C' items in \*(C`fetch\*(C' responses. Returns a string containing the \s-1MIME\s0 type of the message. This is the left-hand portion of a \s-1MIME\s0 media type. For example, the type of \*(C`text/plain\*(C' is \*(C`text\*(C'. Returns a string containing the \s-1MIME\s0 subtype of the message. This is the right-hand portion of a \s-1MIME\s0 media type. For example, the subtype of \*(C`text/plain\*(C' is \*(C`plain\*(C'. Returns a reference to a hash containing the key/value attribute pairs in the \*(C`Content-Type\*(C' field.

If, for example, the \*(C`Content-Type\*(C' field was:

  Content-Type: text/plain; charset=us-ascii

The hash would contain one entry the a key of \*(C`charset\*(C', and a value of \*(C`us-ascii\*(C'. The key is always forced to be lowercase, but the case of the value is retained from the server. Returns the disposition type in the \*(C`Content-Disposition\*(C' field. Returns \*(C`undef\*(C' if no such field exists. Returns a reference to a hash containing the key/value attributer pairs in the \*(C`Content-Disposition\*(C' field. A reference to an empty hash is returned if no such field exists, or if there are no parameters in the field. Returns a reference to a list of the language tags present in the \*(C`Content-Language\*(C' field. Returns a reference to an empty hash if no such field is present.

Multipart

This is a container for \*(C`BodyStructure\*(C' objects that are multipart entities. Returns a list reference of the body parts contained in the multipart entity.

Bodypart

This is a container for singlepart entities in \*(C`BodyStructure\*(C' and \*(C`Multipart\*(C' objects. Return a string containing the contents of the \*(C`Content-ID\*(C' field, if one is present, otherwise returns undef. Return a string containing the contents of the \*(C`Content-Description\*(C' field, if one is present, otherwise returns undef. Returns a string containing the contents of the \*(C`Content-Transfer-Encoding\*(C' field. Returns \*(C`undef\*(C' if no such field is in the entity. Returns the number of octets in the entity. If the \s-1MIME\s0 content type is \*(C`message/rfc822\*(C' or the major type is \*(C`text\*(C', returns the number of lines in the entity, else returns \*(C`undef\*(C'. If the \s-1MIME\s0 content type is \*(C`message/rfc822\*(C', returns a \*(C`Net::IMAP::Envelope\*(C' object, otherwise returns undef. If the \s-1MIME\s0 content type is \*(C`message/rfc822\*(C', returns a \*(C`Net::IMAP::BodyStructure\*(C' object, otherwise returns undef. Returns a string containing the contents of the \*(C`Content-MD5\*(C' field. Returns \*(C`undef\*(C' if no such field is in the entity.

Envelope

This is a container for envelope data in \*(C`fetch\*(C' responses.

For those familiar with \s-1SMTP\s0, this is not the same type envelope. Rather, it is a composite structure containing key source, destination, and reference information in the message. When retrieved from the server, it is populated into a \*(C`Net::IMAP::Envelope\*(C' object. The following methods are available. Returns a string with the contents of the \*(C`Date\*(C' field. Returns a string with the contents of the \*(C`Subject\*(C' field. Returns a list reference of \*(C`Net::IMAP::Addr\*(C' objects with the contents of the \*(C`From\*(C' field. Returns a list reference of \*(C`Net::IMAP::Addr\*(C' objects with the contents of the \*(C`Sender\*(C' field. If no \*(C`Sender\*(C' field is present in the message, the server will default it to the contents of the \*(C`From\*(C' field. Returns a list reference of \*(C`Net::IMAP::Addr\*(C' objects with the contents of the \*(C`Reply-To\*(C' field. If no \*(C`Reply-To\*(C' field is present in the message, the server will default it to the contents of the \*(C`From\*(C' field. Returns a list reference of \*(C`Net::IMAP::Addr\*(C' objects with the contents of the \*(C`To\*(C'field. Will return \*(C`undef\*(C' if no \*(C`To\*(C' field exists in the message. Returns a list reference of \*(C`Net::IMAP::Addr\*(C' objects with the contents of the \*(C`Cc\*(C' field. Will return \*(C`undef\*(C' if no \*(C`Cc\*(C' field exists in the message. Returns a list reference of \*(C`Net::IMAP::Addr\*(C' objects with the contents of the \*(C`Bcc\*(C' field. Will return \*(C`undef\*(C' if no \*(C`Bcc\*(C' field exists in the message. Returns a string with the contents of the \*(C`In-Reply-To\*(C' field. Returns \*(C`undef\*(C' if no such field is present in the message. Returns a string with the contents of the \*(C`Date\*(C' field. Returns \*(C`undef\*(C' if no such field is present in the message.

Addr

This is a container for address structures in \*(C`Envelope\*(C' objects. Returns a string containing the phrase portion of the address, or \*(C`undef\*(C' if no phrase is present. Returns a string containing the route portion of the address, or \*(C`undef\*(C' if no route information is present. Returns a string containing the localpart portion of the address, or \*(C`undef\*(C' if no localpart is present. Returns a string containing the domain portion of the address, or \*(C`undef\*(C' if no domain is present. Returns a string representation of the contents of the object.
