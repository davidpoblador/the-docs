SYNOPSIS

#include <visa.h>

ViStatus viClose(ViObject object);

DESCRIPTION

Closes the handle object and frees the underlying object. Connections to a resource are terminated only when the last session is closed.

RETURN VALUE

On success, VI_SUCCESS is returned.
