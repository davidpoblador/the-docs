DESCRIPTION

Net::NBName::NameQuery represents a decoded NetBIOS name query response.

METHODS

$nq->addresses

Returns a list of ip addresses returned for the queried name. These are returned as a list of \*(C`Net::NBName::NameQuery::RR\*(C' records. Most name queries will only return one ip address, but you will get multiple ip addresses returned for names registered by multihomed hosts or for group name queries.

$nq->ttl

Time to live. This is the lifespan of the name registration.

$nq->\s-1RA\s0

Recursion available. This flag is typically set if the responding host is an \s-1NBNS\s0 server, and can be used to determine if it was an \s-1NBNS\s0 server that responded.

$nq->as_string

Returns the object's string representation.
