SYNOPSIS

  use HTML::WikiConverter;
  my $wc = new HTML::WikiConverter( dialect => 'TikiWiki' );
  print $wc->html2wiki( $html );

DESCRIPTION

This module contains rules for converting \s-1HTML\s0 into TikiWiki markup. See HTML::WikiConverter for additional usage details.

Formatting rule documentation:

* Markup: http://doc.tikiwiki.org/tiki-index.php?page_ref_id=268 * Images: http://doc.tikiwiki.org/tiki-index.php?page_ref_id=277 * Links: http://doc.tikiwiki.org/tiki-index.php?page_ref_id=270

Unsupported formatting rules:

* text boxes
