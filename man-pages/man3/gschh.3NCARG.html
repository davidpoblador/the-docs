gschh (3NCARG)

Sets the character height for text.

SYNOPSIS

CALL GSCHH (CHH)

C-BINDING SYNOPSIS

#include <ncarg/gks.h>

void gset_char_ht(Gdouble char_ht);

DESCRIPTION

CHH

(Real, Input) - Gives the height in world coordinates at which a character is drawn using the GTX output primitive. CHH > 0. By default CHH = 0.01.

ACCESS

To use GKS routines, load the NCAR GKS-0A library ncarg_gks.
