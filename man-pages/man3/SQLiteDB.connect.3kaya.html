SYNOPSIS

SQLiteDB connect( String fname )

ARGUMENTS

fname The filename of the database

DESCRIPTION

Use this function to open the database file (creating it if it does not exist). You can then use the functions from DB(3kaya) to query the database.

  • db = SQLiteDB::connect("site.db"); res = exec(db,"SELECT * FROM pagedata WHERE page_id = 10");
