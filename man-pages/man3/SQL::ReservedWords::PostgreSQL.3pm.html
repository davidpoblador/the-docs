SYNOPSIS

   if ( SQL::ReservedWords::PostgreSQL->is_reserved( $word ) ) {
       print "$word is a reserved PostgreSQL word!";
   }

DESCRIPTION

Determine if words are reserved by PostgreSQL.

METHODS

Returns a boolean indicating if $word is reserved by either PostgreSQL 7.3, 7.4, 8.0 or 8.1. Returns a boolean indicating if $word is reserved by either PostgreSQL 7.3 or 7.4. Returns a boolean indicating if $word is reserved by either PostgreSQL 8.0 or 8.1. Returns a list with PostgreSQL versions that reserves $word.

words

Returns a list with all reserved words.

EXPORTS

Nothing by default. Following subroutines can be exported:

is_reserved
is_reserved_by_postgresql7
is_reserved_by_postgresql8
reserved_by
words
