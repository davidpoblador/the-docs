DESCRIPTION

gactivate_ws (Activate workstation) - activates a GKS workstation.

SYNOPSIS

#include <ncarg/gks.h>

void gactivate_ws(Gint ws_id);

DESCRIPTION

ws_id

(Input) - A number identifying the workstation to be activated. ws_id must be the same as that used in some previous gopen_ws call.

USAGE

Active workstations receive all GKS output primitives.

ACCESS

To use the GKS C-binding routines, load the ncarg_gks and ncarg_c libraries.
