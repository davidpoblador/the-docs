SYNOPSIS

Void open( String identifier=progName(), [LogOption] opts=[LogPid], LogFacility facility=LogUser )

ARGUMENTS

identifier The identifier for logging, conventionally (and by default) the program name.

opts The logging options. The default is to include the process ID with logging.

facility The logging facility to use. The default is LogUser

DESCRIPTION

Opens the new logging handle with the specified options.
