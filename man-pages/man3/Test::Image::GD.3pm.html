SYNOPSIS

  use Test::More plan => 1;
  use Test::Image::GD;

  cmp_image('test.gif', 'control.gif', '... these images should match');

  # or

  my $test = GD::Image->new('test.gif');
  my $control = GD::Image->new('control.gif');
  cmp_image($test, $control, '... these images should match');

  # some other test functions ...

  size_ok('camel.gif', [ 100, 350 ], '... the image is 100 x 350");

  height_ok('test.gif', 200, '... the image has a height of 200');
  width_ok('test.gif', 200, '... the image has a width of 200');

DESCRIPTION

This module is meant to be used for testing custom graphics, it attempts to \*(L"visually\*(R" compare the images, this means it ignores invisible differences like color palettes and metadata. It also provides some extra functions to check the size of the image.

FUNCTIONS

This function will tell you whether the two images will look different, ignoring differences in the order of colors in the color palette and other invisible changes. Both $got and $expected can be either instances of \*(C`GD::Image\*(C' or either a file handle or a file path (both are valid parameters to the \*(C`GD::Image\*(C' constructor).
