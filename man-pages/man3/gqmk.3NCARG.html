gqmk (3NCARG)

Retrieves the current value for the marker type.

SYNOPSIS

CALL GQMK (ERRIND, MTYPE)

C-BINDING SYNOPSIS

#include <ncarg/gks.h>

void ginq_marker_type(Gint *err_ind, Gint *marker_type);

DESCRIPTION

ERRIND

(Integer, Output) - If the inquired value cannot be returned correctly, a non-zero error indicator is returned in ERRIND, otherwise a zero is returned. Consult "User's Guide for NCAR GKS-0A Graphics" for a description of the meaning of the error indicators.

MTYPE

(Integer, Output) - Returns the current value for the polymarker type as set by default or by a call to GSMK.

1

. (dot)

2

+ (plus)

3

* (asterisk) This is the default

4

o (circle)

5

X (cross)

ACCESS

To use GKS routines, load the NCAR GKS-0A library ncarg_gks.
