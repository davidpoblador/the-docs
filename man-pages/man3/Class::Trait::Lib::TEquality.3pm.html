DESCRIPTION

TEquality adds a number of equality testing features, including type-equality as well as object instance equality.

REQUIRES

The \*(C`equalTo\*(C' method is expected to return either true if its two arguments are equal (by whatever standards your devise), or false if they are not.

OVERLOADS

==
!=

PROVIDES

This is the inverse of \*(C`equalTo\*(C'. This will determine type equality, meaning it will determine if both arguements are derived from the same type. This will attempt to discern whether or not the two arguments given are the same object. It even takes into account the possibility that the objects might also have utilized the TPrintable trait, and so works around that automatic stringification.
