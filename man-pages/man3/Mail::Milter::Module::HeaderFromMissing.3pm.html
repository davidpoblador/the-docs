SYNOPSIS

    use Mail::Milter::Module::HeaderFromMissing;

my $milter = new Mail::Milter::Module::HeaderFromMissing();

my $milter2 = &HeaderFromMissing; # convenience

DESCRIPTION

This milter module rejects any message at the \s-1DATA\s0 stage that is missing the From: header. This header should never be absent on any message, even if that message is missing Date: or Subject:.
