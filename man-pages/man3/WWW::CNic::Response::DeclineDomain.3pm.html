SYNOPSIS

        use WWW::CNic;

        my $query = WWW::CNic->new( OPTIONS );
        $query->set( PARAMETERS );

        my $response = $query->execute();

DESCRIPTION

Response module for declining domain renewals via the CentralNic Toolkit (http://toolkit.centralnic.com/). This module inherits all its methods from the base class, WWW::CNic::Response.

METHODS

This class has no additional methods than those it inherits from WWW::CNic::Response. Check WWW::CNic::Response for information about available methods.
