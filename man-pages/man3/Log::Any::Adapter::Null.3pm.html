VERSION

version 0.15

SYNOPSIS

    Log::Any::Adapter->set('Null');

DESCRIPTION

This Log::Any adapter discards all log messages and returns false for all detection methods (e.g. is_debug). This is the default adapter when Log::Any is loaded.
