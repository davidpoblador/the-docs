The update rule for the encoding matrix H. The formula used is \[ H_{a\mu} \leftarrow H_{a\mu} ac{\sum_{i} W_{ia} V_{i\mu}/(WH)_{i\mu}} {\sum_{k} H_{ka}} \] The function takes in all the matrices and only changes the value of the H matrix.

V Input matrix to be factorized.

W Basis matrix.
