SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'ro' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Romanian.

This locale inherits from the DateTime::Locale::root locale.

It contains the following data.
