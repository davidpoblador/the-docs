SYNOPSIS

    [% USE gdc = GD.Constants %]

# --> the constants gdc.gdBrushed, gdc.gdSmallFont, gdc.GD_CMP_IMAGE # are now available

EXAMPLES

[% FILTER null; USE gdc = GD.Constants; USE im = GD.Image(200,100); black = im.colorAllocate(0 ,0, 0); red = im.colorAllocate(255,0, 0); r = im.string(gdc.gdLargeFont, 10, 10, "Large Red Text", red); im.png | stdout(1); END; -%]

DESCRIPTION

The \s-1GD\s0.Constants plugin provides access to the various \s-1GD\s0 module's constants (such as gdBrushed, gdSmallFont, gdTransparent, \s-1GD_CMP_IMAGE\s0 etc). When \s-1GD\s0.pm is used in perl it exports various contstants into the caller's namespace. This plugin makes those exported constants available as template variables.

See Template::Plugin::GD::Image and \s-1GD\s0 for further examples and details.
