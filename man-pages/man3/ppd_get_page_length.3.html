ppd_get_page_length (3)

Set of functions for retrieving the page dimension from a ppd file.

SYNOPSIS

#include <ppd.h>

float ppd_get_page_length(PpdFile *ppd, const char *name);

PpdSize *ppd_get_page_size(PpdFile *ppd, const char *name);

float ppd_get_page_width(PpdFile *ppd, const char *name);

DESCRIPTION

These function provide the programmer with a means of querying the dimensions of a given page layout within a PPD file. All three function accept two parameters as input. The first is a pointer to a valid PpdFile structure (see ppd_file_new(3)), and the second if the name of the page layout to be queried.

RETURN VALUE

ppd_get_page_length()

Returns the page length for a given layout

ppd_get_page_size()

Returns a pointer to the page size record for the given size

ppd_get_page_width()

Returns the page width for a given layout
