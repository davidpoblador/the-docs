SYNOPSIS

#include <firestring.h>

-lfirestring

char *firestring_chug(char *s)

DESCRIPTION

firestring_chug() removes whitespace from the beginning of the string s and returns the result.

RETURN VALUE

Returns s with any leading whitespace removed.
