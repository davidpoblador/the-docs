SYNTAX

#include <stralloc.h>
#include <buffer.h>

int buffer_putsa(buffer* b,stralloc* x);

DESCRIPTION

buffer_putsa is equivalent to buffer_put(b,x.sa,x.len).

RETURN VALUE

0 if everything is fine, -1 on error (setting errno).
