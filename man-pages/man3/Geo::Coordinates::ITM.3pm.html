VERSION

This document describes Geo::Coordinates::ITM version 0.02

SYNOPSIS

  use Geo::Coordinates::ITM qw( ll_to_grid grid_to_ll );

  my ( $lat, $lon ) = grid_to_ll( $east, $north );
  my ( $east, $north ) = ll_to_grid( $lat, $lon );

DESCRIPTION

Convert back and forth between Irish Transverse Mercator grid and \s-1WGS84\s0. The conversion code was stolen wholesale from <http://bit.ly/ZqpUA>.

http://svn.geograph.org.uk/svn/branches/british-isles/libs/geograph \ /conversionslatlong.class.php

Nothing is exported by default. The exportable functions are \*(C`ll_to_grid\*(C' and \*(C`grid_to_ll\*(C'.

INTERFACE

Convert a latitude, longitude (\s-1WGS84\s0) coordinate pair into an \s-1ITM\s0 easting and northing.

my ( $east, $north ) = ll_to_grid( $lat, $lon ); Convert an \s-1ITM\s0 easting, northing pair to a \s-1WGS84\s0 latitude, longitude.

my ( $lat, $lon ) = grid_to_ll( $east, $north );
