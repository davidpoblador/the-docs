SYNOPSIS

gnome-mud [options]

DESCRIPTION

This manual page documents briefly the gnome-mud command.

Gnome-mud is a MUD client written in Gtk+ and GNOME-libs. At the moment, it supports multiple connections, aliases, variables, triggers, ANSI colors, custom keybindings, MCCP and other features. Gnome-mud also has a plugin system similar to zMUD's, so adding new functionality should be easy at user level.

OPTIONS

Generic GNOME and GTK options can be accessed using the --help flag.

-?, --help

Show summary of available options

--usage

Display brief usage message

-V, --version

Print version of gnome-mud

--display=DISPLAY

Specify an X display where gnome-mud should connect to
