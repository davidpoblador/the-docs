SYNOPSIS

cf2bmp file.lev image.bmp

DESCRIPTION

This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manual page.

cf2bmp converts the level file file.lev to an image named image.bmp. The unit and tile sets must be present in the working directory.

OPTIONS

--help Show a help text and exit.
