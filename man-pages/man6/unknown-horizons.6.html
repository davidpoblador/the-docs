SYNOPSIS

unknown-horizons [--load-game savegame] [--start-map mapname] {-d | --debug} [--debug-modules modulename] unknown-horizons [{-h | --help} | {-v | --version}]

DESCRIPTION

This manual page documents briefly the unknown-horizons commands.

unknown-horizons is a real time simulation game

OPTIONS

The program follows the usual GNU command line syntax, with long options starting with two dashes (`-').

--load-game savegame

Loads a saved game. The savegame name matches the displayed string in the ingame menu

--start-map mapname

Loads a available map. The mapname matches the displayed string in the ingame menu

-d, --debug

Enables debug output for Unknwon Horizons

--debug-modules modulename

Activates the logging for the specified submodule

-h, --help

Show summary of options.

FILES

${HOME}/.unknown-horizons/save

Savegame Folder containing all saved games
