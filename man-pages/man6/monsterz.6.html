SYNOPSIS

monsterz [OPTION]...

DESCRIPTION

Options

-h, --help

display this help and exit

-v, --version

display version information and exit

-f, --fullscreen

start in full screen mode

-m, --nomusic

disable music

-s, --nosfx

disable sound effects

--outfd <fd>

output scores to file descriptor <fd>

--data <dir>

set alternate data directory to <dir> --score <file> set score file to <file>
