SYNOPSIS

glpeces [nooptions]

DESCRIPTION

glpeces is a tagram game that runs in X. Tangram is a kind of puzzle game wher payer has to arrange a set of pieces to match a given shape. The program have 33 tangram and more 4600 figures to play. For a complete description, see the program help.

OPTIONS

-v, --version

4.0.
