SYNOPSIS

magicor [OPTION]

DESCRIPTION

Magicor is a small puzzle game similar to the old NES classic solomon's key. You control tux the penguin and create ice blocks to extinguish all fires on each map.

OPTIONS

-h, --help

show this help message and exit

-c CONFIGPATH, --config=CONFIGPATH

use this default config, default /etc/magicor.conf

-j JOYSTICK, --joystick=JOYSTICK

enable/disable joystick

-m MUSIC, --music=MUSIC

enable/disable music

-s SOUND, --sound=SOUND

enable/disable sound

-f FULLSCREEN, --fullscreen=FULLSCREEN

enable/disable fullscreen
