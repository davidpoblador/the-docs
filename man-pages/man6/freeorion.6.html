SYNOPSIS

freeorion [OPTIONS]

DESCRIPTION

FreeOrion is a turn-based space empire and galactic conquest (4X) computer game being designed and built by the FreeOrion project. FreeOrion is inspired by the tradition of the Master of Orion games, but is not a clone or remake of that series or any other game. The game is about the construction of a living, breathing universe in a grand campaign model. You can develop your own galactic empire and explore new star systems with your fleets as one of several unique human or alien species. Colonization of new worlds, researching new technology, managing your resources and defending your planets against the AI or other human players on a galactic scale are your main objectives.

--external-server-address

  • Address to connect to in external server mode. If used, this client becomes the manager of the game. | Default: localhost

--force-external-server

  • Force the client not to start a server, even when hosting a game on localhost, playing single player, etc.

-f, --fullscreen

  • Start the game in fullscreen. Clicking Apply may cause this to take effect, or may require a restart.

--fullscreen-monitor-id

  • Select which monitor to use in fullscreen mode. Primary monitor should be index 0. May require a restart to take effect. | Default: 0

-g, --generate-config-xml

  • Uses default settings, settings from any existing config.xml file, and settings given on the command line to generate a config.xml file. This will overwrite the current config.xml file, if it exists.

-h, --help

  • Print this help message.

--limit-fps

  • Toggles FPS limiting on or off. Limit is set with Max FPS option. Default: 1

--load

  • Loads the specified single-player save game. | Default:

--log-level

  • Sets the level at or above which log messages will be output (levels in order of decreasing verbosity: DEBUG, INFO, NOTICE, WARN, ERROR, CRIT, ALERT, FATAL, EMERG) | Default: DEBUG

--max-fps

  • Toggles FPS limit, if enabled. Limiting is toggled with Limit FPS. Default: 60

--multiplayersetup.host-address

  • Address to connect to when joining a multiplayer game. Default: localhost

--multiplayersetup.player-name

  • Player name to use when hosting or joining a multiplayer game. | Default:

-q, --quickstart

  • Starts a new quick-start game, bypassing the main menu.

-r, --render-simple

  • Sets several map and GUI rendering options to improve frame rate and reduce rendering CPU use. Useful for configuring to run on lower-powered graphics adapters without adjusting each setting separately.

--reset-fullscreen-size

  • Stores whether to reset the stored fullscreen resolution. If false, the stored values are used, but if true, the values are reset to the max the rendering system supports. | Default: 1

--resource-dir

  • Sets the root directory for the game resource files (game content and data files). | Default: \,/usr/share/games/freeorion/default\/

-S, --save-dir

  • The directory in which saved games are saved and from which they are loaded. | Default: /home/apo/.freeorion/save

--show-fps

  • Toggles FPS display on or off. | Default: 0

--stringtable-filename

  • Sets the language-specific string table filename. Default: \,/usr/share/games/freeorion/default/stringtables/en.txt\/

--tech-demo

  • Try out the 3D combat tech demo.

--test-3d-combat

  • Test 3D combat resolution.

--verbose-logging

  • Toggles verbose logging of universe contents and effect evaluation. Default: 0

--verbose-sitrep

  • Toggles inclusion of situation report messages with errors. | Default: 0

--version-string

  • Tracks the FreeOrion version for which config.xml was generated. Config.xml for different versions will be ignored. Default: v0.4.4 [SVN 7640] CMake
