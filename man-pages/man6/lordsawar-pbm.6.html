SYNOPSIS

DESCRIPTION

This manual page documents briefly the lordsawar-pbm command. This manual page was written for the Debian/Ubuntu distributions because the original program does not include them.

lordsawar-pbm e-mails the save game file and turn file to the opposing player.

USAGE

lordsawar-pbm save-game-file turn-file
