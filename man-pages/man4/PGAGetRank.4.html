PGAGetRank (4)

Returns the rank of the processor in communicator comm. if comm is null or a sequential version of pgapack is used, pgagetrank() returns 0.

INPUT PARAMETERS

OUTPUT PARAMETERS

SYNOPSIS

PGAContext *ctx
MPI_Comm comm

EXAMPLE

Example:
PGAContext  *ctx;
int          rank;
:
rank = PGAGetRank(ctx, MPI_COMM_WORLD);
