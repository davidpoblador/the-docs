SYNOPSIS

dfg2tptp <input-file> <output-file>

DESCRIPTION

dfg2tptp is a program which converts a problem input file in \s-1DFG\s0 format into a problem input file in \s-1TPTP\s0 format. The \s-1TPTP\s0 problem format is used by the \s-1TPTP\s0 library of test problems for automated theorem proving, available at http://www.math.miami.edu/~tptp/. Various tools exist to convert problems in \s-1TPTP\s0 format into input files for other theorem provers.
