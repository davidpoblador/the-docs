SYNOPSIS

yagtd [options] <file>

DESCRIPTION

yagtd is invoked with the name of a text file as first and only argument. This file is used to store the to-do list.

yagtd opens an interactive command-prompt for adding, managing and viewing to-do items.

The command mode is described in the yagtd manual. The manual also includes a practical introduction to using yagtd.

OPTIONS

-c, --color

Activate color highlighting

-q, --quiet

Do not print copyright message
