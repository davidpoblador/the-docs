SYNOPSIS

typelib-dump [ module ]

DESCRIPTION

If typelib-dump is called without any arguments, it lists all ORBit2 typelib paths and the typelib modules in them. Otherwise the argument module is taken as the name of a typelib module. This module gets inspected and all types and interfaces of the module as well as all interface methods are dumped to the screen.
