SYNOPSIS

q4wine-helper [KEYs] ...

DESCRIPTION

q4wine-helper is a console utility for Q4Wine which helps to handle Wine application exit status and its stdout\stderr output logging.

OPTIONS

A summary of options is included below.

--prefix

sets current prefix name

--wine-debug

sets WINEDEBUG variable

--console

run with output in console

--display

sets DISPLAY variable

--nice

sets program niceness. See man nice(1) for details

--desktop

sets program virtual desktop resolution

--program-bin

sets program binary

--program-args

sets program args

--wrkdir

sets program working directory

--program-lang

sets program LANG variable

--override

sets WINEDLLOVERRIDES variable

EXIT STATUS

0 if OK or -1 if serious troubles
