SYNOPSIS

debget-madison [switch]... package...

DESCRIPTION

debget-madison shows what versions of a (source or binary) package are available in Debian. It's similar to the madison tool, but it can be used from anywhere on the Internet.

OPTIONS

--debug

Turn debugging on. Specify multiple times for more detail.

--help

Show the usage message and die.

--version

Show the version number and exit.
