Briefly list status of launchpad bugs

DESCRIPTION

lp-list-bugs takes one or more Launchpad bug numbers, and lists the status of each bug in a concise format. For example:

$ lp-list-bugs 3
Bug 3: Custom information for each translation team
  rosetta: Fix Released
  ubuntu: Invalid
  mono (Ubuntu): Invalid

OPTIONS

-h, --help

Display a help message and exit.
