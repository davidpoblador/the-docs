DESCRIPTION

This manual page documents briefly the pypar2 command.

PyPar2 is a graphical frontend for the Linux par2 command line, written in Python. Its GUI uses GTK through the PyGTK library.

PyPar2 is designed to be very easy to use. For this reason: * Advanced settings are present, but hidden by default. * There is no preferences dialog, all selected options are automatically saved and restored.
