SYNOPSIS

rainbow-resume RESUME_UID program-to-execute

DESCRIPTION

This program allow one to resume a precreated isolated UID. It accepts a RESUME_UID, which may be reused between sessions, and the path of a program-to-execute after the container is created.

OPTIONS

This program does not accept any additional options or parameters.
