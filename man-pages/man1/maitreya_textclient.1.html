SYNOPSIS

maitreya_textclient [--help] [--date <str>] [--location <str>] [--vedic] [--western] [--html] [--year <num>] [--month <num>] [--day <num>] [--eclipses] [--hora] [--ephemeris] [--ephemeris-sunrise] [--ephemeris-ingress] [--ephemeris-lunar] [--ephemeris-kpevents <str>] [--file <str>] [--base-data] [--planet-list] [--vedicplanets] [--westernplanets] [--kp] [--bhavas] [--nakshatras] [--vargas] [--ashtakavarga] [--yogas] [--jaimini] [--astronomical] [--arabic] [--printsummary <num>] [--shadbala] [--dasas <num>] [--dasas-long <num>] [--aspects] [--transits] [--solar-arc] [--directions] [--lunar-arc] [--constant-arc] [--solar] [--uranian] [--uranian-yearlypreview] [--file2 <str>] [--partner-vedic] [--partner-composite]

DESCRIPTION

--help

show this help message

--date=<str>

Set UT date YYYY-MM-DD HH:MM:SS

--location=<str>

Set location "name LL:LL BB:BB ZZ"

--vedic

Force Vedic mode (instead of default)

--western

Force western mode (instead of default)

--html

print output in HTML format (instead of text)

--year=<num>

set additional year (transit, Solar)

--month=<num>

set additional month (transit)

--day=<num>

set additional day (transit)

--eclipses

Show eclipses

--hora

Show Hora

--ephemeris

Show ephemeris

--ephemeris-sunrise

Show ephemeris sunrise/Tithi

--ephemeris-ingress

Show ephemeris ingress events

--ephemeris-lunar

Show ephemeris Lunar events

--ephemeris-kpevents=<str>

Show ephemeris kp events for Dasa #n

--file=<str>

Open given file and use data for calculation

--base-data

Show base data of chart

--planet-list

List planets (like in main window, using switch for Vedic/western)

--vedicplanets

Show vedic planets (like in TextView)

--westernplanets

Show western planets (like in TextView)

--kp

Show Krishnamurti Paddhati

--bhavas

Show Bhavas

--nakshatras

Show Nakshatras

--vargas

Show Vargas

--ashtakavarga

Show Ashtakavarga

--yogas

Show Yogas

--jaimini

Show Jaimini calculation

--astronomical

Show astronomical data

--arabic

Show Arabic parts

--printsummary=<num>

Show printout summary nb #n (header of PDF output)

--shadbala

Show Shad Bala

--dasas=<num>

Show Dasa nb #n (short form)

--dasas-long=<num>

Show Dasa nb #n (short form)

--aspects

Show aspects

--transits

Show transits

--solar-arc

Show Solar arc

--directions

Show directions

--lunar-arc

Show Lunar arc

--constant-arc

Show constant arc

--solar

Show Solar (Tajaka) positions

--uranian

Show Uranian Analysis

--uranian-yearlypreview

Show yearly preview (Uranian astrology)

--file2=<str>

Open second file for partner chart

--partner-vedic

Show Vedic partner analysis

--partner-composite

Show partner composite chart
