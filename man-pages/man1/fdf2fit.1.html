SYNOPSIS

fdf2fit [inputfile] [outputfile]

DESCRIPTION

This manual page documents briefly the fdf2fit command.

This manual page was written for the Debian distribution because the original program does not have a manual page.

fdf2fit is a program that converts Origin-4 .fdf fit parameter files to Grace format.

OPTIONS

inputfile

Can be - for standard input.

outputfile

Can be - for standard output. If left blank inputfile will be used but the suffix .fdf will be replaced by .fit .
