d.colorlist (1grass)

Output a list of all available display colors with a configurable separator (default is comma).

KEYWORDS

display, setup

SYNOPSIS

d.colorlist

d.colorlist help

d.colorlist [fs=string] [--verbose] [--quiet]

Parameters:

fs=string

character for separation of list items

Default: ,

DESCRIPTION

d.colorlist report the available color names. The list contains all available display colors with a configurable separator (default is comma).
