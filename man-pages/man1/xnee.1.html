DESCRIPTION

The program suite GNU Xnee can record and replay an X session. GNU Xnee also has the ability to distribute events to multiple displays. Xnee gets copies of X protocol data from the X server. These are either printed to file ( record mode) or replayed and synchronised (replay mode). During record and replay cnee can distribute the record/replayed events to multiple displays.
