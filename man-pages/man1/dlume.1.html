SYNOPSIS

dlume

DESCRIPTION

The Dlume is nice, gtk2-based addressbook. You can easily add, edit and delete records to/from database (but Dlume doesn't rely on an outside database - It stores your contacts in XML format). The Quick-search feature allows you find required entry in confortable way. Export to CSV and HTML formats is also available.

OPTIONS

Dlume has no options.

FILES

${prefix}/bin/dlume

Program executable

$HOME/.dlume/dlume

User's Dlume configuration

$HOME/.dlume/dlume-data.xml

Addressbook data file

$HOME/.dlume/dlume-accels

Keyboard accelerators file

$HOME/.dlume/*.lock

A lock file containing PID who is using the dlume database. This file is removed on normal exit. In case of crash, this file must be deleted manually.
