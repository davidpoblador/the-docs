SYNOPSIS

Do not call this program directly, instead call the symlinks: razor-check razor-report razor-revoke razor-admin

DESCRIPTION

This manual page briefly documents razor-client , the razor metaprogram.

This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manual page. Instead, the subcommands are documented; to access this information, type

man razor-check
man razor-report
man razor-revoke
man razor-admin
