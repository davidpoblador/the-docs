SYNOPSIS

videocut

DESCRIPTION

This manual page documents briefly the videocut binary. videocut is an open source desktop application specialized for creating compositions of screenshots from video files. It using xine-lib and Qt4.

BINARY PARAMETERS

The current release of videocut does not support any kinds of command-line options yet.
