DESCRIPTION

This script selects fastq sequences in a file according to names given in a name file. Given a FASTQ file (and possibly a FASTQ quality file) and a file with names of reads, select the sequences from the input FASTQ (and quality file) and writes them to an output FASTQ.

If fastq quality file is present (same basename, but with .qual appended), then also selects sequences from there.

Please check out the documentation below for more detailed information.

SYNOPSIS

fastqselect <options>

OPTIONS

-infile

name filename containing all fastq sequences

-name

name filename containing all names of sequences to select

-outfile name

filename where to write selcted sequences to
