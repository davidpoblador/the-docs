SYNOPSIS

  stag-autoddl.pl -parser XMLAutoddl -handler ITextWriter file1.txt file2.txt

  stag-autoddl.pl -parser MyMod::MyParser -handler MyMod::MyWriter file.txt

DESCRIPTION

script wrapper for the Data::Stag modules

-help|h

shows this document

-p|parser \s-1FORMAT\s0

\s-1FORMAT\s0 is one of xml, sxpr or itext, or the name of a perl module xml assumed as default

-w|writer \s-1FORMAT\s0

\s-1FORMAT\s0 is one of xml, sxpr or itext, or the name of a perl module This is only used if transforms are required on the data to turn it into relational form; the transformed xml (or other stag format) is stored in the file specified by -t

-link|l \s-1NODE_NAME\s0

this node name will be used as a linking table multiple linking nodes can be set: -l foo2bar -l x2y -l bim2bum

-t \s-1TRANSFORMED_FILE_NAME\s0

the transformed input file is written here
