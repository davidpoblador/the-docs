SYNOPSIS

mate-time-admin [OPTION...]

DESCRIPTION

mate-time-admin is part of the MATE system tools, a set of tools to easily access and manage system configuration.

mate-time-admin allows you to set the time, date and timezone of your system, as well as setting any time server to synchronize your local time server.

OPTIONS

mate-time-admin accepts the standard MATE and GTK options.

AUTHORS
