SYNOPSIS

xdhamma [OPTIONS] [num]

DESCRIPTION

The Dhammapada is an anthology of 423 verses from the Kuddaka Nikaya part of the Theravada Pali Canon. It contains Buddhist ethical and spiritual precepts. One of the translations is by John Richards; the other is an older translation by F. Max Muller.

xdhamma prints a graphical notification using notify\(hysend

OPTIONS

\(hyr | \(hy\(hyrichards (Default)

Use the Richards translation.

\(hym | \(hy\(hymuller

Use the Muller translation.

\(hyp | \(hy\(hypolish

Use the Polish translation (default for Polish locale).

\(hyg | \(hy\(hygrunge

Use the grunge Buddha icon with notify\(hysend.

\(hye | \(hy\(hyenso

Use the enso icon with notify\(hysend.

\(hyl | \(hy\(hylotus

Use the lotus icon with notify\(hysend.

\(hyk | \(hy\(hykdialog

Display Dhammapada verse with kdialog.

\(hyz | \(hy\(hyzenity

Display Dhammapada verse with zenity.

[num]

Print the given number verse.

EXAMPLES

To display a verse each time you log in on a graphical (X Window) session, run xdhamma in your DE session start\(hyup or ~/.xsession.
