SYNOPSIS

hmm2index [options] database

DESCRIPTION

hmm2index is a utility that creates a binary SSI ("squid sequence index" format) index for an HMM database file called database. The new index file is named database.ssi. An SSI index file is required for hmm2fetch to work, and also for the PVM implementation of hmm2pfam.

OPTIONS

-h

Print brief help; includes version number and summary of all options, including expert options.
