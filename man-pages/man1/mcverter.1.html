SYNOPSIS

mcverter [options] <input directory or file(s)>

DESCRIPTION

Required options:

-o [ --output ] arg

Output directory for converted files

-f [ --format ] arg

Output format: fsl, spm, meta, nifti, analyze, or bv

Additional options:

-h [ --help ]

Produce help message

-v [ --verbose ]

Verbose mode

-q [ --quiet ]

Quiet mode

-V [ --version ]

Version info

-j [ --split_subj ]

Save each subject to a separate directory

-x [ --split_dir ]

Save each series to a separate directory

-d [ --fourd ]

Save output volumes as 4D files

-a [ --ho ]

Save header only (metaimage only)

-n [ --nii ]

Save files as .nii (nifti/fsl only)

-b [ --v16 ]

Save .v16 files (BrainVoyager only)

-r [ --rescale ]

Apply rescale slope and intercept to data

-u [ --patientid ]

Use patient id instead of patient name for output file -s [ --skip ] arg (=0) Skip first 'arg' volumes

-m [ --match ] arg

Only convert files whose series descriptions include this string

-F [ --fnformat ] arg

Use this format for name of output file: +/- PatientName, PatientId, SeriesDate, SeriesTime, StudyId, StudyDescription, SeriesNumber, SequenceName, ProtocolName, SeriesDescription
