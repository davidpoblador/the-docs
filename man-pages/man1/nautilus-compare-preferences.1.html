SYNOPSIS

nautilus-compare-preferences

DESCRIPTION

'nautilus-compare-preferences' is a graphical tool to set preferences for Nautilus Compare which is a small extension for the Nautilus file manager, which allows users to compare selected files from the context menu as well as remember them for later comparison.

nautilus-compare-preferences has been written in Python and GTK+, and after installing the application you can run it from the application launcher.
