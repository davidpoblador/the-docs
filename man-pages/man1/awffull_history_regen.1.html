DESCRIPTION

Usage:

  • awffull_history_regen.pl [options]

  • NB! Must have the GNU Date command!

OPTIONS

--help

Print a brief help message and exit.

--man

Print the manual page and exit. --dir directory

  • The directory to use, looking for old webalizer usage_YYYYMM.html files. If not present will use the current directory. --date gnu-date-location

  • This program requires the GNU date command, use this option, if it's in a non-standard place.
