SYNOPSIS

vibrate [\|--help\|]

vibrate program [\|arguments\|]

DESCRIPTION

vibrate runs the specified program, which presumably uses the NCBI C toolkit, with the Vibrant library preloaded. The main effect of this preloading is that omitting arguments produces a dialog box for setting parameters rather than a usage message.

OPTIONS

--help

Print usage message and exit.
