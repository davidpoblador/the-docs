SYNOPSIS

sidcon <options> [--psid|--sidplay] <files>

DESCRIPTION

sidcon can convert your SID tunes between psid and sidplay format. Please note that you don't have to convert between different formats, sidplay can play tunes anyway.

OPTIONS

-a

assume yes (y) on all questions.

-c

only check each file's loadaddress.

-r

process current directory.

-d

recurse into current directory tree.
