SYNOPSIS

music-librarian [dirs]

DESCRIPTION

This manual page documents the music-librarian command. music-librarian is a GUI program to manage a collection of music. It manipulates a "library": one or more music files stored underneath one or more directories. If it is started with arguments, they indicate the library to open; otherwise, no library will be opened initially.
