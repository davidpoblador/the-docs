VERSION

2.2.1

DESCRIPTION

Push a file to a LEGO Mindstorms NXT brick

USAGE

Usage: nxt_push [--host <macaddress>] <file>

OTHER PROGRAMS

This package also provides nxt_server and nxt_test. See their documentation for a description of their functions. There are also example files and the python nxt module. The nxt file manager is in the python-nxt-filer package.
