SYNOPSIS

vlpatch [options] original patch

DESCRIPTION

This manual page documents briefly the vlpatch command.

vlpatch is a program that applies a patch to an original with the VCS::Lite perl module.

OPTIONS

--output outfile

If --output is not specified, the patched file is put in place of the original, and the original is renamed to *.orig.
