SYNOPSIS

crosshurd

DESCRIPTION

crosshurd Sets up a functioning Debian GNU/Hurd system on another partition by using apt to fetch the packages.

FILES

/etc/crosshurd/source.list

The sources.list to use for fetching the .deb files.
