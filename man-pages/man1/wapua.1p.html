SYNOPSIS

wApua [-f configfile] [-d debuglevel] [starturl]
wApua (-h|--help|--usage)

DESCRIPTION

wApua is a browser for \s-1WAP\s0 (Wireless Application Protocol) pages written in the Wireless Markup Language (\s-1WML\s0). It supports \s-1WML\s0 1.1 and 1.2 except forms.

It is written in Perl and uses the Perl/Tk library for its \s-1GUI\s0 and libwww-perl for the network parts. So it supports all transport protocols, libwww-perl (\s-1LWP\s0) supports.

OPTIONS

-h, --help, --usage

Shows a summary of options.

-v, --version

Shows the version of wApua.

-d n, --debug=n

Sets the level of debug output to n.

CONFIGURATION

The system-wide default configuration can be changed in wApua::Config

Per-user configuration can be done by creating a .wApuarc file on Linux and other \s-1UNIX\s0 derivatives or clones or a wApua.ini file on Windows and MacOS 9 in your home directory.

See the example configuration wApua.rc which comes with wApua.
