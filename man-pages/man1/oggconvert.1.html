SYNOPSIS

oggconvert

DESCRIPTION

OggConvert is a small Gnome utility which uses GStreamer to convert (almost) any media file to the patent-free Ogg Vorbis, Theora and Dirac formats.

This version of OggConvert is GUI-only and does not have a command-line interface.

OPTIONS

--help-gst

Prints a list of all available GStreamer debugging switches
