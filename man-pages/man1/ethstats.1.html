SYNOPSIS

ethstats

Description

The ethstats utility is a quick way to display network interface statistics. After invoking the command the following information will appear every 10 seconds for each interface and a line with totals of all interfaces:

eth0: 1.00 Mb/s In 1.15 Mb/s Out - 1032.0 p/s In 2300.0 p/s Out

The data shown is in and out speeds in megabits per second and in and out counts for packets per second. This data is obtained by comparing /proc/net/dev each sampling. (Therefore, this tool is Linux specific)
