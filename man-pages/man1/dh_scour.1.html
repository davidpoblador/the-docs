SYNOPSIS

dh_scour [debhelper options] [-- params]

DESCRIPTION

dh_scour is a debhelper program that will run the scour \s-1SVG\s0 image scrubber on all shipped \s-1SVG\s0 files.

OPTIONS

-- params

Pass \*(L"params\*(R" to scour\|(1). By default, no extra arguments are passed.
