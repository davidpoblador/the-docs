ldaptor-find-server (1)

Find ldap servers for a given list of distinguished names

  4. ldaptor-find-server: Find ldap servers for a given list of distinguished names

SYNOPSIS

ldaptor-find-server DN..

DESCRIPTION

ldaptor-find-server searches the DNS for the name of an LDAP server responsible for the given list of distinguished names (DNs).

RESULT

ldaptor-find-server prints the priority, weight, host name and port of each LDAP server.
