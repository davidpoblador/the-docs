SYNOPSIS

xttitle title [ icon ]

DESCRIPTION

This manual page documents briefly the xttitle command. This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manual page.

xttitle is a program that changes the title (and possibly icon) of your X terminal emulator windows. It should work for any program that emulates an xterm-like terminal under X.

You may want to create an alias in your shell for 'cd', so that your window title keeps track of your current working directory. In Bash, the following seems to work for me:

	update_title()
	{
		[ $TERM = 'xterm ] && xttitle "[$$] ${USER}@${HOSTNAME}:$PWD"
	}

	function cd()
	{
		builtin cd $*; update_title
 	}

You can then set up other things that run update_title - for instance you may want to put it in your shell startup file so that the title gets updated when you open a new terminal emulator window.
