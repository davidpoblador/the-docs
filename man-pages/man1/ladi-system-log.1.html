DESCRIPTION

LADITools is a collection of tools aiming to achieve the goals of the LADI project to improve desktop integration and user workflow of GNU/Linux audio system based on JACK and LASH. Those tools take advantage of the D-Bus interfaces recently added to JACK and LASH to ease the configuration and use of those two great softwares.

The following tools are included:

g15ladi

A JACK monitor for g15 keyboards.

ladi-control-center

Graphical interface to setup JACK's configuration.

ladi-player

Compact front end to that allows users to start, stop and monitor a LADI system

ladi-system-log

A JACK, LASH and a2jmidid log viewer.

ladi-system-tray

System tray icon that allows users to start, stop and monitor JACK, as well as start some JACK related applications.

wmladi

Window Maker dockapp which uses a menu similar to ladi-system-tray's.
