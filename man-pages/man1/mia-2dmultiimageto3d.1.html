SYNOPSIS

mia-2dmultiimageto3d -i <in-file> -o <out-file> [options]

DESCRIPTION

mia-2dmultiimageto3d This program is used to convert a 2D multi-image file to a 3D image.

OPTIONS

-i --in-file=(input,required)

input images For supported file types see PLUGINS:2dimage/io

-o --out-file=(output,required)

output image For supported file types see PLUGINS:3dimage/io
