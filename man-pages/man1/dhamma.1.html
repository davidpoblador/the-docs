SYNOPSIS

display\(hydhammapada [OPTIONS] [num]

dhamma [OPTIONS]

DESCRIPTION

The Dhammapada is an anthology of 423 verses from the Kuddaka Nikaya part of the Theravada Pali Canon. It contains Buddhist ethical and spiritual precepts. One of the translations is by John Richards; the other is an older translation by F. Max Muller.

display\(hydhammapada and dhamma both print to the standard output.

xdhamma prints using notify\(hysend

See man xdhamma for details

OPTIONS

\(hyr

Use the Richards translation (default).

\(hym

Use the Muller translation.

\(hyb

Display corresponding verses from both translations.

\(hypl

Use the Polish translation (default for Polish locale).

[num]

Print the given number verse.

all

Print all the verses.

With no arguments a random verse is printed.

EXAMPLES

To display a verse each time you log in on a console or open a terminal, put display\(hydhammapada in your ~/.profile, ~/.bash_profile or ~/.login, as appropriate for your shell.

To display a verse each time you log in on a graphical (X Window) session, run xdhamma in your DE session start\(hyup or ~/.xsession.

To use display\(hydhammapada with cowsay

display\(hydhammapada | cowsay \(hyW 120 \(hyt

To use display\(hydhammapada with zenity

display\(hydhammapada | zenity \(hy\(hytext\(hyinfo \(hy\(hytitle="Dhammapada Verse" \(hy\(hywidth=600 \(hy\(hyheight=200

KDE has "amor" toy that can display various tips.  To use the dhammapada for that enter these commands at shell prompt:


mkdir ~/.kde/share/apps/amor

display\(hydhammapada \(hyr all |perl \(hy0777 \(hype 's/\n\s*\n\s*\n/\n%\n/g' >~/.kde/share/apps/amor/tips

Then start amor K\(hy>Toys\(hy>Amor, right click on amor animation, select "Options" and check "Show random tips".

One can automatically attach Dhammapada verse in e-mail footer. To do this with mutt add set signature=display-dhammapada| in your ~/.muttrc file.
