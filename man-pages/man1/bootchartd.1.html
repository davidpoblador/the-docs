SYNOPSIS

bootchartd [start|stop]

DESCRIPTION

bootchartd is a daemon collecting data from boot process.

Please see bootchart2 (1) for more information.

OPTIONS

start

Start the daemon.

stop

Stop the daemon

FILES

/etc/bootchartd.conf

/sbin/bootchartd
