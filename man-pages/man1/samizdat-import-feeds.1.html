Update rss feeds imported by hosted samizdat sites

SYNOPSIS

samizdat-import-feeds

DESCRIPTION

samizdat-import-feeds Fetches all RSS feeds configured for all Samizdat sites specified under /etc/samizdat. Uses standard Ruby RSS library to parse the feeds, results are stored in persistent DRb cache in a form usable by Samizdat engine.

OPTIONS

Accepts no options, operates based on configuration files under /etc/samizdat.

FILES

/etc/samizdat/sites.yaml

/etc/samizdat/sites/*.yaml
