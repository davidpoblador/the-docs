SYNOPSIS

kumolog [options] <logfile.mpac>

DESCRIPTION

Convert binary logs into human-readable format.

OPTIONS

-f, --follow

  same as `tail -f'

-t, --tail

  show only last  N logs (default: N=10)

-h, --head

  show only first N logs (default: N=10)

-n, --lines=[-]N

  specify N

EXAMPLE

$ kumolog -f svr1_log.mpac

$ kumolog -h -n 50 svr1_log.mpac
