SYNOPSIS

poldi-ctrl [options] command

DESCRIPTION

This manual page documents briefly the poldi-ctrl commands.

poldi-ctrl is used to see information related to pam_poldi

OPTIONS

A summary of options is included below. For a complete description, see the Info files.

-d, --dump

Dump certain card information

-k, --print-key

Print authentication key from card

-s, --print-serialno

Print serial number from card

--debug

Enable debugging mode
