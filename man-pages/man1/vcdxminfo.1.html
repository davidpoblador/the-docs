SYNOPSIS

vcdxminfo [OPTION...]

DESCRIPTION

-i, --generic-info

show generic information

-a, --dump-aps

dump APS data

--relaxed-aps

relax APS constraints

-o, --output-file=FILE

file for XML output

-p, --progress

show progress

-v, --verbose

be verbose

-q, --quiet

show only critical messages

--gui

enable GUI mode

-V, --version

display version and copyright information and exit

Help options:

-?, --help

Show this help message

--usage

Display brief usage message
