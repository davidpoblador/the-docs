SYNOPSIS

cuba-partview N M [...] < file

DESCRIPTION

cuba-partview reads Cuba's verbose output from stdin and displays the N-M plane of the tessellation on screen. Each pair of dimensions is shown in a separate tab.

To use this utility you should pass verbosility flag to Cuba's routines, and that flags should be set to 3. With verbosility set to 3 the program will print to stdout data, that is used as input for cuba-partview .
