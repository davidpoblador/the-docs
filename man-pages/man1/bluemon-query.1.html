SYNOPSIS

bluemon-query [<address>]

DESCRIPTION

This manual page documents briefly the bluemon-query command.

bluemon-query sends queries to the bluemon(1) program over DBUS.

QUERIES

running bluemon-query with no arguments will cause it to query for the first connected device. Giving the parameter of a bluetooth ID will query for that ID. It will print the ID and signal strength if connected.
