SYNOPSIS

vorbistagedit file1 [\|file2\|[\|file3...\|]\|]

vorbistagedit [\|--version|-V|-v\|]

vorbistagedit [\|--help|-h\|]

DESCRIPTION

vorbistagedit allows batch editing of vorbis comments with an editor. If more than one OGG Vorbis file is specified, vorbistagedit opens a unique editor for all files given. The supported file extensions for vorbistagedit are .oga, .ogg, .ogv, .ogx, and .spx.

OPTIONS

-v, -V, --version

Show the version of vorbistagedit.

-h, --help

Show a short help message.

RELATED TO vorbistagedit…

vorbiscomment(1), ogginfo(1)

ENVIRONMENT

EDITOR

Defines the default editor. If it's not defined, then sensible-editor(1) is used.
