matroxset (1)

Switch the output mode of matrox video cards

SYNOPSIS

   matroxset [OPTIONS] [VALUE]

OPTIONS

-c

Get/set control value.

-e

Edit controls interactively.

-f \s-1DEVICE\s0

Manipulate the frame buffer \s-1DEVICE\s0 (default is /dev/fb1).

-l

List controls

-m

Connect this frame buffer to the outputs specified in \s-1VALUE\s0 (0 is none, 1 is primary, 2 is secondary, 4 is digital flat panel; these values may be summed for multiple outputs).

-o \s-1OUTPUT\s0

Set \s-1OUTPUT\s0 (0 is primary, 1 is secondary, default is 1) to the output mode \s-1VALUE\s0 (128 is monitor, 1 is \s-1PAL\s0 \s-1TV\s0, 2 is \s-1NTSC\s0 \s-1TV\s0).

-p

Display information about blanking.

DESCRIPTION

Output Matrox card video to television. \s-1PAL\s0 50Hz and \s-1NTSC\s0 59.94Hz are supported, Composite and S-Video output.

Contains two demo programs, normal and swapped. First selects \s-1CRTC1\s0 on first output and \s-1CRTC2\s0 on second one. Swapped connects \s-1CRTC1\s0 to secondary head and \s-1CRTC2\s0 to primary head. You can use these programs if you need text mode or 8bpp mode on \s-1TV\s0, as \s-1CRTC2\s0 is incapable of these modes.

In program call, If \s-1VALUE\s0 is specified, set the specified option to \s-1VALUE\s0. If \s-1VALUE\s0 is not specified, retrieve the value of the specified option.

For technical information, see TV-out on the G450 at <http://www.bglug.ca/matrox_tvout>. See also \*(C`Matrox Marvel\*(C' tips for G200, G400 at <http://marvel.sourceforge.net/>. Matrox frame buffer headers at /usr/include/linux/matroxfb.h
