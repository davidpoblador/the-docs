SYNOPSIS

widgetserver [options]

DESCRIPTION

QtOctave is a Qt front-end to GNU Octave, and widgetserver is a tool for adding widgets to QtOctave. GNU Octave is a high-level language, highly compatible with MATLAB, and primarily intended for numerical computations. GNU Octave provides a convenient command line interface for solving linear and nonlinear problems numerically.
