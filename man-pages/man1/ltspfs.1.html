SYNOPSIS

ltspfs

DESCRIPTION

This manual page documents briefly the ltspfs command.

ltspfs is a program that mounts fuse based filesystems that got exported by ltspfsd

OPTIONS

Usage: ltspfs host:/dir/to/mount /mountpoint [fuse options]
