SYNOPSIS

ieee-copyout [--clean] [--version]

DESCRIPTION

This manual page documents briefly the ieee-copyout commands. This manual page was written for the Debian distribution because the original program does not have a manual page.

ieee-copyout is used for creating an archive of the format that the IEEE wants for paper submissions. This script is part of LaTeX-Mk.

OPTIONS

--clean

Delete *.log files.

--version

Show version of program.
