SYNOPSIS

gm-print_doc DOCUMENT_TYPE FILE(S) ...

DESCRIPTION

gm-print_doc is a helper script which GNUmed uses to print documents.

OPTIONS

DOCUMENT_TYPE

The type of document to print. GNUmed will use this option to indicate that the documents to print are of a certain logical type. This script can use this tag to decide what to do about a particular print job.

Example: medication_list

FILE(S)

Those are the files which GNUmed wants printed. Note that they can be of any mimetype as long as the target printing system knows how to deal with them.

Typically, though, they will be PDF files.
