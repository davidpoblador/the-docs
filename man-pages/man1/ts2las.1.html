SYNOPSIS

ts2las \kx [-h | -i arg [-o arg] [-v] [Filtering options] ]

DESCRIPTION

ts2las converts TerraSolid .bin files into LAS files with optional output filters.

OPTIONS

ts2las options:

\*(T<-i\*(T> arg, \*(T<--input\*(T> arg

input TerraSolid .bin file

\*(T<-o\*(T> arg, \*(T<--output\*(T> arg
