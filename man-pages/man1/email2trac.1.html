email2trac (1)

Creates and amends trac tickets from e-mail.

SYNOPSIS

email2trac [-h] [--help] [-f configuration file] [--file=configuration file] [-n] [--dry-run] [-p project name] [--project=project name] [-t name] [--ticket_prefix=name]

DESCRIPTION

email2trac creates tickets for the issue tracking system Trac from e-mail. The default configuration file is /etc/email2trac.conf.

Is also amends existing tickets, if the subject contains the ticket number. The syntax for the ticket number in the e-mail subject line is #NNN:, i.e. hash mark, ticket number, colon.
