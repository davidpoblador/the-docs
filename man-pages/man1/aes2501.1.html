SYNOPSIS

aes2501 [-r]

DESCRIPTION

This manual page documents briefly the aes2501 command.

aes2501 is a command line program that scans fingerprints from an aes2501 scanner. It creates an image in local directory named fp.pnm, so assure you have write permissions in the directory you exec the command.

The AES 2501 fingerprint scanner vendor is Authentec and this sensor can be found in:

•

Medion MD85264 USB sensor

•

HP nx6125 notebook

•

HP Compaq nx6320 notebook

•

Fujitsu-Siemens P7120 notebook.

•

LG P1 PRO Express Dual notebook.

Please, if you have this device and It's not in this list, send us to include it. (xerakko -at- debian -dot- org)

OPTIONS

-r Scan in reverse mode
