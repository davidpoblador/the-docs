SYNOPSIS

gaphorconvert [ options ] file1 file2...

DESCRIPTION

This manual describes gaphorconvert command.

OPTIONS

-h, --help

Show help message and exit.

-v, --verbose

Verbose output.

-u, --use-underscores

Use underscores instead of spaces for output filenames.

-d directory, --dir=directory

Output to directory.

-f format, --format=format

Output file format, default is pdf.

-r regex, --regex=regex

Process diagrams with name matching regex regular experssion. Diagram name includes package name, regular expressions are case insensitive.
