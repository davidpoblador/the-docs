SYNOPSIS

gnupod_check [OPTION]...

DESCRIPTION

Check for lost/zombie files on the iPod. Use this tool if your OS crashed while uploading songs to the iPod or if you have too much time ;)

--help

display this help and exit

--version

output version information and exit

-m, --mount=DIRECTORY

iPod mountpoint. Default is $IPOD_MOUNTPOINT
