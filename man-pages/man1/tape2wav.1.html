SYNOPSIS

tape2wav [ -r sample rate ] infile outfile

DESCRIPTION

tape2wav converts audio files to ZX Spectrum tape images.

OPTIONS

-r sample rate

causes sample rate to be used as the sample rate for the target wav file.

infile

specifies the tape file to be converted. This file can be in any of the tape formats supported by libspectrum(3).

outfile

specifies the output wav file, if set to “-” will write the file to standard output.
