SYNOPSIS

terrasync [-d path] [-p port] [-v]

DESCRIPTION

FlightGear TerraSync is a utility that automatically downloads the newest version of the FlightGear scenery while the simulator is running. For more information visit <http://wiki.flightgear.org/TerraSync>.

OPTIONS

-d path

Specify the path where to store the scenery data.

-p port

Specify the UDP port where terrasync listens for a FlightGear Flight Simulator connection.

-v

Enable verbose output.
