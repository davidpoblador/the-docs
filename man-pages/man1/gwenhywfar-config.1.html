SYNOPSIS

gwenhywfar-config [options]

DESCRIPTION

This manual page documents briefly the gwenhywfar-config command.

gwenhywfar-config is a little program to output installation paths of Gwenhywfar.

OPTIONS

--includes

gives you the include flags

--libraries

gives you the library flags

--bindir

returns the folder for binary tools

--headers

gives you the location of the header files

--vmajor

gives the major version of LibGwenhywfar

--vminor

gives the minor version of LibGwenhywfar

--vpatchlevel

gives the patchlevel of LibGwenhywfar

--vbuild

gives the build number of LibGwenhywfar

--vtag

gives the tag of LibGwenhywfar (cvs, beta or stable)

--vstring

returns a version string
