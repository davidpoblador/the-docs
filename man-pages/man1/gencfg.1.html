SYNOPSIS

gencfg <devices> <address> <spacing> <Aoff> <Boff> <Dataoff> <intack> <vector> <clock> [hdwe] [param[]

DESCRIPTION

If you have used your scc card with a NET/NOS program onder DOS before, you can use gencfg to generate a configuration file from your autoexec.net. Just type gencfg to read instructions.
