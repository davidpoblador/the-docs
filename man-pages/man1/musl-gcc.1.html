SYNOPSIS

musl-gcc [OPTION...] FILENAME...

DESCRIPTION

musl-gcc acts as a wrapper arround gcc(1) which sets the correct system include paths for linking with musl libc library.

Since musl-gcc is a simple wrapper arround gcc(1) it uses the same command line arguments.

FILES

/usr/lib/$ARCH-linux-musl/musl-gcc.specs musl-gcc spec file that defines new/overwritten include paths.
