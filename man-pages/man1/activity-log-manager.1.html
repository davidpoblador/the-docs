SYNOPSIS

zeitgeist-global-privacy [OPTIONS]

DESCRIPTION

activity-log-manager is a tool for easily setting up blacklists, temporarily stopping all logging and accessing other privacy-related controls related to your activity log.

OPTIONS

The program follows the usual GNU command line syntax, with options starting with two dashes (`--'). A summary of options is included below.

--version

Shows the version number and exits.

-h, --help

Shows a short help message listing all supported options and exits.
