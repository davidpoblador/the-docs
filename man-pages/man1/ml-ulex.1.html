SYNOPSIS

ml-ulex [--dot] [--dump] [--match] [--ml-lex-mode] [--minimize] file

DESCRIPTION

This manual page documents briefly the ml-ulex command. This manual page was written for the Debian distribution because the original program does not have a manual page. The ml-ulex script is part of the SML/NJ CM (Compilation and Library Manager) that is used to generate standalone programs.

OPTIONS

A summary of some of the options is included below.

--dot

Generate DOT output (for graphviz).

--match

Enter interactive matching mode.

--ml-lex-mode

Operate in ml-lex compatibility mode.

RELATED TO ml-ulex…

sml(1), ml-antlr(1), ml-lex(1), ml-yacc(1).
