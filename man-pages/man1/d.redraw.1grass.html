KEYWORDS

display

SYNOPSIS

d.redraw

d.redraw help

d.redraw [-ar] [--verbose] [--quiet]

Flags:

-a

Redraw all frames

-r

Do not preserve individual regions when redrawing all frames

--verbose

Verbose module output

--quiet

Quiet module output

DESCRIPTION

d.redraw redraws the content of the currently selected monitor. The active monitor can be selected with d.mon.
