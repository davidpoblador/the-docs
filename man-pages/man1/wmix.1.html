SYNTAX

wmix [-h]

wmix -f <file>

DESCRIPTION

Dockapp mixer for OSS. Allows toggling record source, muting individual channels, adjusting volume and balance, all in a compact dockapp size, with TV-like on-screen-display for volume levels.

Supports mousewheel to adjust current channelvolume and can be controlled remotely with SIGUSR1 / SIGUSR2 to adjust the volume. Keeps a simple configuration file for user-defined settings.

OPTIONS

-h Display list of command-line options.

-f <file> use alternate config file
