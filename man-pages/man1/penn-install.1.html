SYNOPSIS

penn-install [OPTION]

DESCRIPTION

This manual page documents briefly the penn-install command. It was written for the Debian distribution because the original program does not have a manual page.

penn-install is a program that installs the pennmush server in the user directory from which it is ran. It provides all necessary data and configuration files for each user account it is setup for, as well as the restart script needed to initiate an instance of the pennmush server process.

penn-install itself should only be run once per user to install the necessary files and links. All updating of the pennmush server should be done on a global level by the system administrator.

OPTIONS

These programs follow the usual GNU command line syntax, with long options starting with two dashes (`-'). A summary of options is included below.

-h, --help

display this help and exit

-v, --version

output version information and exit
