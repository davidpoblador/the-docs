SYNOPSIS

nmapsi4-logr

DESCRIPTION

This manual page documents briefly the nmapsi4-logr command. This manual page was written for the Debian distribution because the original program does not have a manual page.

nmapsi4 is a full-featured interface for the nmap(1) security network scanner.

OPTIONS

nmapsi4-logr supports no commandline options. Everything is done via the GUI.
