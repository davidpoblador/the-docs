SYNOPSIS

fsmv [OPTION]... [SOURCE] [DESTINATION]

OPTIONS

-h, --help

show this help message and exit

--debug

Show debug information

-v, --verbose

make output verbose

--listopeners

list all FS openers

--fs=OPENER

import an FS opener e.g --fs foo.bar.MyOpener

-p, --progress

show progress

-t THREAD_COUNT, --threads=THREAD_COUNT

number of threads to use
