SYNOPSIS

iv [options] files...

DESCRIPTION

The iv program is a great interactive image viewer. Because iv is built on top on OpenImageIO, it can display images of any formats readable by ImageInput plugins on hand.

OPTIONS

--help

Print help message

-v

Verbose status messages

-F

Foreground mode
