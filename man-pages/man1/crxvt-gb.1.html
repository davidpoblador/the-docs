SYNOPSIS

krxvt [options] [-e command [args]]

crxvt [options] [-e command [args]]

grxvt [options] [-e command [args]]

DESCRIPTION

krxvt, crxvt, and grxvt are versions of rxvt that have been compiled with support for non-latin character sets. krxvt supports Japanese (kanji) characters; crxvt-big5 supports Chinese (big5) characters; crxvt-gb supports Chinese (GB) characters; and grxvt supports greek keyboards. rxvt is a color vt102 terminal emulator intended as an xterm(1) replacement for users who do not require features such as Tektronix 4014 emulation and toolkit-style configurability. As a result, rxvt uses much less swap space – a significant advantage on a machine serving many X sessions.

More complete description of rxvt and its derivatives can be found in the rxvt(1) manual page.
