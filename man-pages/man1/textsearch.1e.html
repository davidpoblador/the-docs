SYNOPSIS

DESCRIPTION

textsearch is a command line program from EMBOSS (“the European Molecular Biology Open Software Suite”). It is part of the "Data retrieval:Text data" command group(s).

OPTIONS

Input section

-sequence seqall

Required section

-pattern string

The search pattern is a regular expression. Use a | to indicate OR. For example: human|mouse will find text with either 'human' OR 'mouse' in the text

Additional section

-casesensitive boolean

Default value: N

Output section

-html boolean

Default value: N

-only boolean

This is a way of shortening the command line if you only want a few things to be displayed. Instead of specifying: '-nohead -noname -nousa -noacc -nodesc' to get only the name output, you can specify '-only -name' Default value: N

-heading boolean

Default value: @(!$(only))

-usa boolean

Default value: @(!$(only))

Default value: @(!$(only))

-name boolean

Default value: @(!$(only))

-description boolean
