SYNOPSIS

mkgmapgui

DESCRIPTION

mkgmapgui is a graphical user interface for mkgmap.

Although mkgmap has a lot of features, this GUI allows you to do the most common actions: convert an Osm file into an Img file and combine multiple Img files into one big Img file.
