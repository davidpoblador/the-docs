SYNOPSIS

srda-landscape [options]

OPTIONS

-h, --help

show this help message and exit

-d FILE, --data=FILE

data - required

-s, --standardize

standardize data

-n, --normalize

normalize data

-k K

k for k-fold cross validation

-c SETS PAIRS

sets and pairs for monte carlo cross validation

-S, --stratified

for stratified cv

-m MIN, --min=MIN

min value for alpha parameter [default -6]

-M MAX, --max=MAX

max value for alpha parameter [default 6]

-p STEPS, --steps=STEPS

steps for alpha parameter [default 13]

-e SCALE, --scale=SCALE

scale for alpha parameter: 'lin' or 'log' [default log]

-l, --lists

Canberra distance indicator

-a, --auc

Wmw_auc metric computation
