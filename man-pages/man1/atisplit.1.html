SYNOPSIS

atisplit

DESCRIPTION

The program atisplit is a small utility to convert gatos capture file (gatos.yuv) into a series of yuv frames good for input to mpeg_encode program.

mpeg_encode can be obtained on metalab (http://www.metalab.unc.edu/).

note that since mpeg_encode uses yuv files with only quarter of cb and cr fields the output files have less information than the input one. in particular the total size of the output files should be just below 3/4 of the size of the input file.

USAGE

- Start the "TV in a Window" application:

./xatitv &

  and capture whatever you want,

atisplit

  atisplit furiously prints out frames.

- You must edit the default.lparam file (see below);

- And,

mpeg_encode default.lparam

  this produce output.mpg MPEG file.

MPEG_ENCODE CONFIGURATION FILE

You can find a sample default.lparam file below:

#
# use this as a guideline for any parameters you
# don't really understand or don't care about.
#

PATTERN         IBBPBBPBBPBBPBB
OUTPUT          output.mpg

BASE_FILE_FORMAT        YUV
GOP_SIZE                10
SLICES_PER_FRAME        1

PIXEL                   HALF
RANGE                   10
PSEARCH_ALG             LOGARITHMIC
BSEARCH_ALG             CROSS2
IQSCALE                 8
PQSCALE                 10
BQSCALE                 25

REFERENCE_FRAME         ORIGINAL

#
# you really need to understand the following
#
YUV_SIZE                320x240
INPUT_CONVERT           *

INPUT_DIR               ./

INPUT
tv*.yuv                 [000001-000100]
END_INPUT
