SYNOPSIS

marco-theme-viewer [THEME]

DESCRIPTION

marco-theme-viewer allows you to preview any installed Marco theme.

When designing a new Marco theme, you can use marco-theme-viewer to benchmark the performance of a window frame, and to preview it around different types of windows in various mock scenarios.

USAGE

Usage of marco-theme-viewer is fairly simple.

Just call marco-theme-viewer followed by the Name of any valid Marco or Metacity theme installed in "~/.themes" or "/usr/share/themes".

It is case-sensitive.

EXAMPLE

marco-theme-viewer Crux

Preview the Crux theme if it is installed.

FILES

~/.themes

user themes directory

/usr/share/themes

system themes directory

/usr/share/themes/*/metacity-1/metacity-theme-1.xml

OR

/usr/share/themes/*/marco-1/marco-theme-1.xml

theme specification files
