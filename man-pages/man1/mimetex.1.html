SYNOPSIS

mimetex [options] [expression]

DESCRIPTION

This manual page documents briefly the mimetex command. This manual page was written for the Debian distribution because the original program does not have a manual page.

mimetex is a program that parses LaTeX math expressions emmiting directly anti-aliased GIF images without intermediate dvi-to-gif conversion.

-d

dump gif image on stdout [default]

-eexport_file

write gif image to export_file

-finput_file

read expression from input_file

-mmsglevel

verbosity of debugging output

-o

render image with opaque background

-sfontsize

default fontsize, 0-5

-m0-99

controls verbosity level for debugging output (usually used only while testing code).
