SYNOPSIS

aeskeyfind [OPTION] MEMORY-IMAGE

DESCRIPTION

aeskeyfind is a tool that illustrates automatic techniques for locating 128-bit and 256-bit AES keys in a captured memory image.

OPTIONS

-vverboseoutput--printstheextendedkeysandtheconstraintsontherowsofthekeyschedule

-qdon'tdisplayaprogressbar

-tTHRESHOLDsetsthemaximumnumberofbiterrorsallowedinacandidatekeyschedule

-hdisplaysusagehelp
