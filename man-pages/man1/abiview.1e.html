DESCRIPTION

abiview is a command line program from EMBOSS (“the European Molecular Biology Open Software Suite”). It is part of the "Data retrieval:Sequence data,Display" command group(s).

OPTIONS

Input section

-infile infile

Output section

-outseq seqout

-graph xygraph

-startbase integer

-endbase integer

Last sequence base to report or display. If the default is set to zero then the value of this is taken as the maximum number of bases.

-separate boolean

Default value: N

-yticks boolean

Default value: N

-sequence boolean

Default value: Y

-window integer

Default value: 40

-bases string

Default value: GATC
