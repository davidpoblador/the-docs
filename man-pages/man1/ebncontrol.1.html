SYNOPSIS

ebncontrol [option...] sub-command

OPTIONS

-c FILE --configuration-file FILE

specify a configuration file (default: /etc/ebnetd/ebnetd.conf)

-h --help

display this help, then exit

-v --version

display version number, then exit

Sub-commands:

kill

kill ndtpd (send SIGKILL)

restart

restart ndtpd (send SIGHUP)

status

output status of ndtpd

terminate

terminate ndtpd (send SIGTERM)

Default value used in a configuration file:

work-path

/var/run/ebnetd/ebnetd
