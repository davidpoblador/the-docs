SYNOPSIS

kid [options] file [args]

DESCRIPTION

Expand a Kid template file.

options: -e enc, --encoding=enc

  • Specify the output character encoding. Default: utf-8 -o outfile, --output=outfile

  • Specify the output file. Default: standard output -s host:port, --server=host:port

  • Specify the server address if you want to start the HTTP server. Instead of the Kid template, you can specify a base directory. -h, --help

  • Print this help message and exit. -V, --version

  • Print the Kid version number and exit.

file:

  • filename of the Kid template.

args:

  • key=value or other arguments passed to the template.
