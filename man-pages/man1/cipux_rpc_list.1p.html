VERSION

cipux_rpc_list - version 3.4.0.9

SYNOPSIS

 cipux_rpc_list

ABSTRACT

This is the CipUX XML-RPC list for the CipUX::Task \s-1API\s0. The list provides CipUX commands to \s-1RPC\s0 clients.

OPTIONS

-c

Same as --cfg

--cfg

Specifies the configuration.

-D

The same as --debug

debug

h

The same as --help

help

l

The same as --list

list

Lists.

p

The same as --pretty

pretty

Pretty output.

V

The same as --version.

version

Prints the version.

REQUIREMENTS

CipUX CipUX::RPC::Server Getopt::Long Pod::Usage Carp

DESCRIPTION

+----------------------------------------+ | xml rpc command | +========================================+ | cipux_task_create_room | | cipux_task_create_student_account | | cipux_task_create_teacher_account | | cipux_task_destroy_room | | cipux_task_destroy_student_account | | cipux_task_destroy_teacher_account | | cipux_task_list_netgroups | | cipux_task_list_rooms | | cipux_task_list_student_accounts | | cipux_task_list_teacher_accounts | | cipux_task_optain_room_network_address | +----------------------------------------+

USAGE

cipux_rpc_list [OPTIONS]

EXAMPLE

cipux_rpc_list

cipux_rpc_list --list

cipux_rpc_list --debug
