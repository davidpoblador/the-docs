SYNOPSIS

Execute pyragua in a terminal to start the application.

DESCRIPTION

pyragua(1) is a development environment to manipulate Python code designed by members of Semillero de Investigación Pyrox at the Universidad Tecnológica de Pereira.

pyragua(1) is written in Python, a multiplatform language. Pyragua is Free Software and the code can be downloaded in pyragua project page:

http://pyragua.pyroxdev.org
