DESCRIPTION

The dbconfig configuration file controls the configuration of a GNATS database. Each database has its own individual copy of the file, which is located in the gnats-adm administrative subdirectory of the database.

For further information about the dbconfig file, please see the GNATS manual, Keeping Track: Managing Messages With GNATS (also installed as the GNU Info file gnats.info)
