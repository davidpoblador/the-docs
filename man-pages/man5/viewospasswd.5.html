DESCRIPTION

/etc/viewospasswd contains one line for each user account aiming to use kmview-umview as his/her user account. Each line contains two fields delimited by colons (:):

\h'-04'•\h'+03'login name

\h'-04'•\h'+03'command line of the user's startup script

The startup script/command should start a shell as its last action using an exec.
