ethers (5)

Zuordnung von Ethernetadressen nach IP-Adressen

  1. Linux Man Pages
  2. File formats and conventions
  3. ethers

BESCHREIBUNG

/etc/ethers enthält 48 bit Ethernetadressen und die dazu equivalenten IP-Nummern, eine Zeile pro IP-Nummer.

Ethernet-adresse IP-Nummer

Diese zwei Felder d\(:urfen durch eine beliebige Anzahl von SPACE- oder TAB-Zeichen getrennt sein. Hinter einem #-Zeichen am Anfang der Zeile beginnt ein Kommentar, der sich bis zum Ende der Zeile erstreckt. Die Ethernet-adresse wird als .IR x : x : x : x : x : x geschrieben, wobei x eine hexadecimale Zahl zwischen 0 und ff ist, die ein Byte der Adresse repräsentiert, die in Netzwerksbyteorder (big-endian) ist. Die IP-Nummer kann ein Hostname, der \(:uber DNS resolviert werden kann oder eine IP-Adresse in in Punktnotation sein.

08:00:20:00:61:CA pal
