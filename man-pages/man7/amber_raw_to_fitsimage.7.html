SYNOPSIS

esorex amber_raw_to_fitsimage [OPTIONS] FILE.sof

DESCRIPTION

The main purpose of this recipe is to convert the imaging sections of a amber raw files into a fits-cube and a fits-images showing the 5 channels.

This recipe is able to process all amber raw files with a DATA column in the IMAGING_DATA extension, i.e. most of the amber raw files. Therefore no special classification tag needs to be given to the SOF. Moreover, the SOF should include only one amber raw file.

OPTIONS

Note that it is possible to create a configuration file containing these options, along with suitable default values. Please refer to the details provided by the 'esorex --help' command.
