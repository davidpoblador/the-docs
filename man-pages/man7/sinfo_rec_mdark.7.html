Master dark and hot pixels mask generation.

SYNOPSIS

esorex sinfo_rec_mdark [OPTIONS] FILE.sof

DESCRIPTION

This recipe perform raw sinfo_dark data reduction.

The input files are raw sinfo_dark images Their associated tags should be DARK.

The output are a master sinfo_dark (PRO.CATG=MASTER_DARK) and a hot pixels bad pixel map (PRO.CATG=BP_MAP_HP)

OPTIONS

--gen-overpar <bool>

Overwrite DRS ini parameters: (bool; default: True). The full name of this option for the EsoRex configuration file is sinfoni.general.overwrite_parameters [default = True].

--lc_sw <bool>

Correct for bad lines introduced by instrument software: (bool; default: False). The full name of this option for the EsoRex configuration file is sinfoni.general.lc_sw [default = False].

--lc_kappa <long>

Kappa sigma value (long; default: 18). The full name of this option for the EsoRex configuration file is sinfoni.general.lc_kappa [default = 18].

--lc_filt_rad <long>

Filtering radii applied during median filter. Should be small (long; default: 3). The full name of this option for the EsoRex configuration file is sinfoni.general.lc_filt_rad [default = 3].

--bp_noise-thresh_sigma_fct <float>

Threshold Sigma Factor: If the mean noise exceeds this threshold times the clean standard deviation of the clean mean the corresponding pixels are declared as bad (float; default: 10.0). The full name of this option for the EsoRex configuration file is sinfoni.bp_noise.thresh_sigma_factor [default = 10.0].

--bp_noise-lo_rej <float>

low_rejection: percentage of rejected low intensity pixels before averaging (float; default: 10.0). The full name of this option for the EsoRex configuration file is sinfoni.bp_noise.low_rejection [default = 10.0].

--bp_noise-hi_rej <float>

high_rejection: percentage of rejected high intensity pixels before averaging (float; default: 10.0). The full name of this option for the EsoRex configuration file is sinfoni.bp_noise.high_rejection [default = 10.0].

--dark-lo_rej <float>

lower rejection (float; default: 0.1). The full name of this option for the EsoRex configuration file is sinfoni.dark.low_rejection [default = 0.1].

--dark-hi_rej <float>

higher rejection (float; default: 0.1). The full name of this option for the EsoRex configuration file is sinfoni.dark.high_rejection [default = 0.1].

--dark-qc_ron_xmin <long>

qc_ron_xmin (long; default: 1). The full name of this option for the EsoRex configuration file is sinfoni.dark.qc_ron_xmin [default = 1].

--dark-qc_ron_xmax <long>

qc_ron_xmax (long; default: 2048). The full name of this option for the EsoRex configuration file is sinfoni.dark.qc_ron_xmax [default = 2048].

--dark-qc_ron_ymin <long>

qc_ron_ymin (long; default: 1). The full name of this option for the EsoRex configuration file is sinfoni.dark.qc_ron_ymin [default = 1].

--dark-qc_ron_ymax <long>

qc_ron_ymax (long; default: 2048). The full name of this option for the EsoRex configuration file is sinfoni.dark.qc_ron_ymax [default = 2048].

--dark-qc_ron_hsize <long>

qc_ron_hsize (long; default: 4). The full name of this option for the EsoRex configuration file is sinfoni.dark.qc_ron_hsize [default = 4].

--dark-qc_ron_nsamp <long>

qc_ron_nsamp (long; default: 100). The full name of this option for the EsoRex configuration file is sinfoni.dark.qc_ron_nsamp [default = 100].

--dark-qc_fpn_xmin <long>

qc_fpn_xmin (long; default: 1). The full name of this option for the EsoRex configuration file is sinfoni.dark.qc_fpn_xmin [default = 1].

--dark-qc_fpn_xmax <long>

qc_fpn_xmax (long; default: 2047). The full name of this option for the EsoRex configuration file is sinfoni.dark.qc_fpn_xmax [default = 2047].

--dark-qc_fpn_ymin <long>

qc_fpn_ymin (long; default: 1). The full name of this option for the EsoRex configuration file is sinfoni.dark.qc_fpn_ymin [default = 1].

--dark-qc_fpn_ymax <long>

qc_fpn_ymax (long; default: 2047). The full name of this option for the EsoRex configuration file is sinfoni.dark.qc_fpn_ymax [default = 2047].

--dark-qc_fpn_hsize <long>

qc_fpn_hsize (long; default: 2). The full name of this option for the EsoRex configuration file is sinfoni.dark.qc_fpn_hsize [default = 2].

--dark-qc_fpn_nsamp <long>

qc_fpn_nsamp (long; default: 1000). The full name of this option for the EsoRex configuration file is sinfoni.dark.qc_fpn_nsamp [default = 1000].
