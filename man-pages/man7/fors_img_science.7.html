SYNOPSIS

esorex fors_img_science [OPTIONS] FILE.sof

DESCRIPTION

Input files:

  DO category:               Type:       Explanation:             Number:
  SCIENCE_IMG                Raw         Science image               1
  MASTER_BIAS                FITS image  Master bias                 1
  MASTER_SKY_FLAT_IMG        FITS image  Master sky flat field       1

Output files:

  DO category:               Data type:  Explanation:
  SCIENCE_REDUCED_IMG        FITS image  Reduced science image
  PHOT_BACKGROUND_SCI_IMG    FITS image  Reduced science image background
  SOURCES_SCI_IMG            FITS image  Unfiltered SExtractor output
  OBJECT_TABLE_SCI_IMG       FITS table  Extracted sources properties

OPTIONS

--magsyserr <float>

Systematic error in magnitude (float; default: 0.01). The full name of this option for the EsoRex configuration file is fors.fors_img_science.magsyserr [default = 0.01].

--sex_exe <str>

SExtractor executable (str; default: \'/usr/bin/sex\'). The full name of this option for the EsoRex configuration file is fors.fors_img_science.sex_exe [default = /usr/bin/sex].

--sex_config <str>

SExtractor configuration file (str; default: \'/usr/share/cpl- plugins/fors-5.0.0/config/fors.sex\'). The full name of this option for the EsoRex configuration file is fors.fors_img_science.sex_config [default = /usr/share/cpl-plugins/fors-5.0.0/config/fors.sex].

--sex_mag <str>

SExtractor magnitude (str; default: \'MAG_APER\'). The full name of this option for the EsoRex configuration file is fors.fors_img_science.sex_mag [default = MAG_APER].

--sex_magerr <str>

SExtractor magnitude error (str; default: \'MAGERR_APER\'). The full name of this option for the EsoRex configuration file is fors.fors_img_science.sex_magerr [default = MAGERR_APER].

--sex_radius <long>

Background error map median filter radius (unbinned pixels) (long; default: 64). The full name of this option for the EsoRex configuration file is fors.fors_img_science.sex_radius [default = 64].

Note that it is possible to create a configuration file containing these options, along with suitable default values. Please refer to the details provided by the 'esorex --help' command.
