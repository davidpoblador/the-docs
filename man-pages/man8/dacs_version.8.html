Display dacs version information

SYNOPSIS

dacs_version [\m[blue]dacsoptions\m[]\s-2\u[1]\d\s+2]

DESCRIPTION

This program is part of the DACS suite.

The dacs_version web service displays version information about DACS and other third-party packages that have been compiled into DACS. If XML output is selected, a document conforming to \m[blue]dacs_version.dtd\m[]\s-2\u[2]\d\s+2 is returned.

Note

This program is also available as a DACS utility, \m[blue]dacsversion(1)\m[]\s-2\u[3]\d\s+2. Because \m[blue]dacs_admin(8)\m[]\s-2\u[4]\d\s+2 provides the same functionality and more, dacs_version may be removed in a future release.

OPTIONS

Web Service Arguments

The \m[blue]standard CGI arguments\m[]\s-2\u[5]\d\s+2 are understood.

DIAGNOSTICS

The program exits 0 if everything was fine, 1 if an error occurred.
