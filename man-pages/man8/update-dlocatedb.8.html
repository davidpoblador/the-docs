SYNOPSIS

update-dlocatedb

DESCRIPTION

update-dlocatedb generates a plain text package contents listing dlocate from the files /var/lib/dpkg/info/*.list This program is usually called from cron.daily.

FILES

/etc/default/dlocate

Configuration file with default options. Currently the only option is to enable/disable gzip compression of the dlocatedb file.

/var/lib/dpkg/info Directory containing dpkg file lists

/var/lib/dlocate/dlocatedb

Plain text listing of all files belonging to all packages (optionally compressed)
