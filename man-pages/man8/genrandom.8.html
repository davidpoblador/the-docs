genrandom (8)

generate a file containing random data

  1. Linux Man Pages
  2. System administration commands
  3. genrandom

SYNOPSIS

genrandom [-n number] {size} {filename}

DESCRIPTION

genrandom generates a file or a set of files containing a specified quantity of pseudo-random data, which can be used as a source of entropy for other commands on systems with no random device.

ARGUMENTS

-n number In place of generating one file, generates number (from 2 to 9) files, appending number to the name.

size The size of the file, in kilobytes, to generate.

filename The file name into which random data should be written.
