SYNOPSIS

askPrinter [dev-path-to-printer]

DESCRIPTION

This manual page documents briefly the askPrinter command.

This manual page was written for the Debian GNU/Linux distribution because the original program does not have a manual page.

askPrinter is a helper program to help you configure Mtink.

OPTIONS

These programs follow the usual GNU command line syntax, with long options starting with two dashes (`-'). A summary of options is included below.

Options for askPrinter

dev-path-to-printer

Specify the device file associated with the printer. Depending on what kind of printer you have, it could be /dev/parport, /dev/usb/lp0...
