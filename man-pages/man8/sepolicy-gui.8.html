sepolicy-gui (8)

Graphical user interface for selinux policy.

  1. Linux Man Pages
  2. System administration commands
  3. sepolicy-gui

SYNOPSIS

Common options

sepolicy gui [-h ] [ -d DOMAIN ]

DESCRIPTION

Use sepolicy gui to run a the graphical user interface, which allows you to explore how SELinux confines different process domains.

OPTIONS

-h, --help

Display help message

-d, --domain

Initialize gui to the selected domain.
