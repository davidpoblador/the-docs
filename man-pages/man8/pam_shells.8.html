pam_shells (8)

Pam module to check for valid login shell

  1. Linux Man Pages
  2. System administration commands
  3. pam_shells

SYNOPSIS

pam_shells.so

DESCRIPTION

pam_shells is a PAM module that only allows access to the system if the users shell is listed in /etc/shells.

It also checks if /etc/shells is a plain file and not world writable.

OPTIONS

This module does not recognise any options.

MODULE TYPES PROVIDED

The auth and account module types are provided.

RETURN VALUES

PAM_AUTH_ERR Access to the system was denied.

PAM_SUCCESS The users login shell was listed as valid shell in /etc/shells.

PAM_SERVICE_ERR The module was not able to get the name of the user.

EXAMPLES

auth  required  pam_shells.so
