DOCUMENTATION

zg-policy-rc.d is a script which provides flexible handling of the policy-rc.d script. As invoke-rc.d only looks for policy-rc.d in /usr/sbin, putting local binaries there would be a policy violation.

Hence, this script.

If the environment variable \s-1POLICYRCD\s0 is set, the script the variable pointed to is used. If \s-1POLICYRCD\s0 is not set, /etc/zg-policy-rc.d.conf is sourced. /etc/zg-policy-rc.d.conf can set the \s-1POLICYRCD\s0 variable.

If invocation of /etc/zg-policy-rc.d.conf does not give a \s-1POLICYRCD\s0 value set, the script checks for executeable files in /usr/local/sbin/policy-rc.d and /etc/policy-rc.d and uses the first one it finds.
