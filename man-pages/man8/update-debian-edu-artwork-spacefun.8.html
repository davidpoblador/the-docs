SYNOPSIS

update-debian-edu-artwork-spacefun \kx { configure | remove }

DESCRIPTION

update-debian-edu-artwork-spacefun configure enables the Debian Edu Spacefun theming (as shipped with Debian Edu wheezy) system-wide (several login managers, KDE splash, KDE theme, background for all desktops, MATE theme and panel layout, GRUB boot loader, etc.pp.).

update-debian-edu-artwork-spacefun remove disables the Debian Edu themes and switches back to Debian's default themes (from package desktop-base).
