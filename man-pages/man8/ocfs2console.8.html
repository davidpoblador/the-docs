SYNOPSIS

ocfs2console

DESCRIPTION

ocfs2console is a GUI front-end for managing OCFS2 volumes. This tool is especially recommended to configure the O2CB cluster. One can also use this tool to format, tune, mount and umount OCFS2 volumes.
