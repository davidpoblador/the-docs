NOM

validlocale - Vérifier si un ensemble donné de paramètres régionaux est disponible

SYNTAXE

validlocale <locale>

DESCRIPTION

Tester si l'ensemble de paramètres régionaux donné en argument est valable. S'il ne l'est pas, afficher sur la sortie standard la chaîne de caractères à ajouter à /etc/locale.gen pour permettre à locale-gen de générer l'ensemble de paramètres (s'il existe).

FICHIERS

/us/sbin/validlocale

/usr/share/i18n/SUPPORTED

VARIABLES D'ENVIRONNEMENT

DEFAULTCHARSET

Indique l’encodage de caractères qui doit être supposé si l'ensemble de paramètres régionaux donné n'est pas dans la liste des ensembles gérés.

EXEMPLES

Si un ensemble de paramètres régionaux valable est indiqué en paramètre, la sortie d'erreur standard affiche :

  • % validlocale C

    locale 'C' valid and available

Lorsqu'un ensemble de paramètres régionaux non valable (non créé ou inexistant) est indiqué, un message sur la sortie d'erreur l’indique, et une chaîne de caractères à ajouter à /etc/locale.gen pour le créer est envoyée sur la sortie standard :

  • % validlocale fr_MQ@euro

    locale 'fr_MQ@euro' not available

    fr_MQ@euro ISO-8859-1
