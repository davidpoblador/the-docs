SYNTAX

vps [options]

DESCRIPTION

vps - The vps utility is simply a wrapper for the ps program, that enables ps to show you the all the processes on the system, instead of only the ones in the current s_context (see cat /proc/self/status).

OPTIONS

See man ps.

FILES
