SYNOPSIS

DESCRIPTION

gtkorphan is a graphical tool which scans your debian system, looking for orphaned libraries. It implements a \s-1GUI\s0 front-end to deborphan, but adds the package removal capability.

A detailed documentation on the program can be found at: http://www.marzocca.net/linux/gtkorphan.html.
