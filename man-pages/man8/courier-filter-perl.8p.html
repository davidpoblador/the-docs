SYNOPSIS

filterctl [ start | stop ] [ courier-filter-perl ]

DESCRIPTION

The courier-filter-perl executable is a global mail filter for the Courier \s-1MTA\s0 and acts as glue between Courier's courierfilter interface and the Courier::Filter module framework.

For an architectural and administrative overview of the Courier::Filter framework, see Courier::Filter::Overview.

FILES

/etc/courier/filters/courier-filter-perl.conf

This is the configuration file for \*(C`courier-filter-perl\*(C' and Courier::Filter. See Courier::Filter::Overview for its format.
