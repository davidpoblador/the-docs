SYNTAX

sesman [ --nodaemon | --kill | --help ]

DESCRIPTION

sesman is xrdp(8) session manager.

It manages user sessions by authenticating the user and starting the appropriate Xserver

OPTIONS

-n, --nodaemon

Starts sesman in foreground instead of starting it as a daemon.

-k, --kill

Kills running sesman daemon.

-h, --help

Output help information and exit.

FILES

${SESMAN_BIN_DIR}/sesman
