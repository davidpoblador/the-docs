nvdump (8)

Nvdump program to dump nvidia tv-out encoder registers

SYNOPSIS

nvdump [-ar2] [-i [incr]] <from> <to> ...

nvdump

-s <addr> <val> <val> ...

DESCRIPTION

nvdump dumps parts of the Nvidia TV-OUT encoder register space (besides the CRTC) and can be used for troubleshooting. You can use it if you suspect there might be register changes that are to output with 'nvtv -P'

OPTIONS

-h -? --help

print help message

-a

print all

-2

print also PCRTC2 and PRAMDAC2

-r

print registers

-i

set increment

-s

set register values
