SYNOPSIS

dtc_change_bsd_kernel VPS_NUMBER MEMORY_SIZE normal|install

DESCRIPTION

dtc_change_bsd_kernel.sh This shell script is a part of the dtc-xen package that is to be used by the dtc panel to manage a Xen VPS server.

This script generate a Xen startup file with the parameters given to the script.
