SYNOPSIS

tomoyo-editpolicy-agent listen_ip:listen_port

DESCRIPTION

This program is an agent for tomoyo-editpolicy, tomoyo-loadpolicy, tomoyo-savepolicy, tomoyo-pstree, tomoyo-queryd, and tomoyo-auditd.

It listens on the specified \s-1IP\s0 address at the specified port.

Do not run this program unless you know what you are doing. It allows anonymous users to edit policy, as no user authentication is performed. It is designed for systems with resource limitations that make it difficult to run the \s-1TOMOYO\s0 Linux userspace tools. Do not use this for normal desktops or servers.

Before this program can be invoked, you must register it in /sys/kernel/security/tomoyo/manager.

EXAMPLES

Listen to 192.168.1.1 at port 10000 
  tomoyo-editpolicy-agent 192.168.1.1:10000
