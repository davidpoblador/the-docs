SYNOPSIS

mkelfImage [--command-line=command line] [--kernel=path to vmlinux] [--ramdisk=path to ramdisk] [--output=file] [--ramdisk-base=<start addr>]

DESCRIPTION

mkelfImage is a program that makes a elf boot image for linux kernel images. The image should work with any i386 multiboot compliant boot loader, an \s-1ELF\s0 bootloader that passes no options, a loader compliant with the linuxBIOS elf booting spec or with the linux kexec kernel patch. A key feature here is that nothing relies upon \s-1BIOS\s0 calls, but they are made when necessary. This is useful for systems running linuxbios.

BUGS

Not all kernel parameters can be passed with the multiboot image format. ip configuration is not automatically passed to a node. The ramdisk base is hard coded to 8MB by default. This man page need to be updated.
