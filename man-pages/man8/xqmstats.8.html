xqmstats (8)

Display xfs quota manager statistics from /proc

  1. Carta.tech

SYNOPSIS

/usr/sbin/xqmstats

DESCRIPTION

xqmstat queries the kernel for the XFS Quota Manager dquot statistics. It displays:

[bu]

Reclaims

[bu]

Missed reclaims

[bu]

Dquot dups

[bu]

Cache misses

[bu]

Cache hits

[bu]

Dquot wants

[bu]

Shake reclaims

[bu]

Inact reclaims

OPTIONS

None.

quotastats(1), quota(1).
